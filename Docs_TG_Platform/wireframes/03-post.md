```
┌─────────────────────────────────────────────────────────────────┐
│ Лента / Название поста    [← Назад] [↑ К посту] [📋][💬] [•••]  │
├─────────────────────────────────────────────────────────────────┤
│ ┌─────────────────────────────────────────────────────────────┐ │
│ │ [🖼 медиа поста]                                            │ │
│ │ [ТЕКСТ ПОСТА]                                               │ │
│ │ ● Опубликован  👁 5100 · ❤ 87 · ↗ 23   [✏ Редактировать]    │ │
│ └─────────────────────────────────────────────────────────────┘ │
│                                                                 │
│ [Я] ...                                                         │
│ [ИИ] ...                                                        │
│                                                                 │
│ ┌─────────────────────────────────────────────────────────────┐ │
│ │ Написать сообщение...                                       │ │
│ │ [+] [🧠 GPT-4.1 ▾] [🔎 Web ▾]                          [↑]  │ │
│ └─────────────────────────────────────────────────────────────┘ │
└─────────────────────────────────────────────────────────────────┘
```

## Шапка

- Слева — хлебные крошки `Лента / Название поста`.
- Справа в одной группе слева направо: `← Назад`, `↑ К посту` (появляется, когда первое сообщение ушло выше области видимости), `📋 Заметки`, `💬 Чаты`, меню `•••`.
- `📋 Заметки` и `💬 Чаты` — переключатели режимов с подсветкой активного состояния; повторный клик по активной кнопке возвращает в режим `chat`.
- Кнопка `← Назад` работает по стеку переходов внутри пространства поста: переключение режимов и открытие конкретного чата кладёт состояние в стек, `← Назад` сначала разматывает стек и только при пустом стеке уходит на ленту.
- Шапка реализована в стиле liquid glass: фиксирована поверх скролла, контент уходит под неё с эффектом «жидкого стекла» (адаптируется под светлую/тёмную тему).

## Локальный чат

- При входе в пространство поста активного чата нет (`currentPostChatId === null`), composer пуст, показывается только карточка поста.
- Первое отправленное сообщение создаёт новый локальный чат (заголовок — первые ~40 символов сообщения). AI-ответ приходит в этот же чат.
- Дальше работа идёт в нём, пока пользователь не переключится через `Чаты` или не выберет `Новый чат`.
- Память между разными локальными чатами одного поста не делится.

## Состояние B — Inline-редактирование

По кнопке `✏ Редактировать` карточка поста разворачивается в редактор:

- сверху остаётся TG-стиль медиа-блок;
- под ним появляется кнопка `🖼 Добавить медиа` (открывает file picker, добавляет `PostMedia` к медиа поста);
- ниже — textarea с текстом поста, без `min-height` и `resize`: высота равна исходному размеру карточки и растёт по контенту;
- внизу — кнопки `Отмена` и `Сохранить`.

## Состояние C — Режим заметок

Режим включается кнопкой `📋 Заметки` в шапке.

Показывается сетка локальных заметок:

- клик по карточке → страница заметки;
- `● ИИ / ○ ИИ` на карточке;
- `🗑` удаление;
- `+` новая заметка.

## Состояние D — Список локальных чатов

Режим включается кнопкой `💬 Чаты` в шапке.

```
┌─────────────────────────────────────────────────────────────────┐
│ Лента / Название поста    [← Назад] [📋][💬*] [•••]             │
├─────────────────────────────────────────────────────────────────┤
│                          [+ Новый чат]                          │
│                                                                 │
│ ┌─────────────────────────────────────────────────────────┐     │
│ │ Перепиши пост            30 апр 2026             [🗑]   │     │
│ │ "Сделай его более..."                                   │     │
│ └─────────────────────────────────────────────────────────┘     │
│ ┌─────────────────────────────────────────────────────────┐     │
│ │ Подбор заголовков        28 апр 2026             [🗑]   │     │
│ │ "Дай 5 вариантов..."                                    │     │
│ └─────────────────────────────────────────────────────────┘     │
└─────────────────────────────────────────────────────────────────┘
```

- Карточка локального чата: название (первое сообщение), превью последнего сообщения, дата.
- `+ Новый чат` — пустой черновик нового локального чата, мгновенно переключает в режим `chat`.
- Клик по карточке открывает соответствующий чат в режиме `chat`.
- `🗑` удаляет локальный чат.

## Меню `•••`

### Для опубликованного поста

- `📝 Новый чат`
- `🗑 Удалить пост`

### Для черновика

- `📝 Новый чат`
- `📢 Опубликовать`
- `🕐 Запланировать`
- `🗑 Удалить пост`

### Для отложенного

- `📝 Новый чат`
- `📢 Опубликовать`
- `✕ Отменить публикацию`
- `🗑 Удалить пост`

## Composer (режим `chat`)

| Элемент | Действие |
|---|---|
| Textarea | Авто-ресайз вверх до 16 строк, дальше скролл |
| `+` | Attach-меню (раскрывается вверх): `Прикрепить пост`, `Медиа из поста` (медиа текущего поста), `Прикрепить файл с компьютера`, `Медиа из прикреплённых постов` (когда привязаны другие посты). Подменю — миниатюры 72×72 с адаптивной сеткой и собственным скроллом |
| `🧠 …` | LLM-`ModelPicker` (раскрывается вверх) |
| `🔎 …` | Web Search `ModelPicker`; рядом — флаг `Мультиответ`, если в профиле включён `Включить в мультиответ` у нескольких моделей |
| `↑` | Отправить сообщение в активный локальный чат (или создать новый, если активного нет) |
