# Пространство поста (`post`)

## Состояние A — Чат (по умолчанию)

```
┌─────────────────────────────────────────────────────────────────┐
│ Лента / Название поста            [← Назад] [↑ К посту] [•••]  │
├─────────────────────────────────────────────────────────────────┤
│ [ПЕРВОЕ СООБЩЕНИЕ: ТЕКСТ ПОСТА]                                │
│ ● Опубликован  👁 5100 · ❤ 87 · ↗ 23      [✏ Редактировать]   │
│                                                                 │
│ [Я] ...                                                         │
│ [ИИ] ...                                                        │
│                                                                 │
│ [📎]  Написать сообщение...                               [↑]   │
└─────────────────────────────────────────────────────────────────┘
```

## Состояние B — Inline-редактирование

По кнопке `✏ Редактировать` первое сообщение превращается в `textarea` с кнопками `Отмена` и `Сохранить`.

## Состояние C — Режим заметок

Режим включается через меню `•••` пунктом `📋 Заметки`.

Показывается сетка локальных заметок:

- клик по карточке -> страница заметки;
- `● ИИ / ○ ИИ` на карточке;
- `🗑` удаление;
- `+` новая заметка.

## Меню `•••`

### Для опубликованного поста

- `📋 Заметки` (или `💬 Чат` в режиме заметок)
- `🗑 Удалить`

### Для черновика

- `📋 Заметки` (или `💬 Чат`)
- `📢 Опубликовать`
- `🕐 Запланировать`
- `🗑 Удалить`

### Для отложенного

- `📋 Заметки` (или `💬 Чат`)
- `📢 Опубликовать`
- `✕ Отменить публикацию`
- `🗑 Удалить`

## Поле ввода

| Иконка | Действие |
|---|---|
| `📎` | Прикрепить медиа из поста или файл с компьютера |
| `↑` | Отправить сообщение в локальный чат |
