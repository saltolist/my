## Хлебные крошки

```
Лента / Два года...                    [← Назад]  [•••]
```

Шапка состоит **только** из строки хлебных крошек — никаких дополнительных строк со статусом или кнопками.

- Путь: **Лента** → название поста (сокращается если длинное)
- [← Назад] справа — возвращает в ленту
- [•••] рядом с [← Назад] — кнопка контекстного меню

## Меню [•••] пространства поста

Состав меню зависит от статуса поста:

**Опубликованный пост:**
```
                          ┌───────────────────────┐
                          │ ↑ К посту             │
                          │ 📋 Заметки            │
                          │ ✏ Редактировать пост  │
                          │ 🗑 Удалить             │
                          └───────────────────────┘
```

**Черновик:**
```
                          ┌───────────────────────┐
                          │ ↑ К посту             │
                          │ 📋 Заметки            │
                          │ ✏ Редактировать пост  │
                          │ 📢 Опубликовать        │
                          │ 🕐 Запланировать       │
                          │ 🗑 Удалить             │
                          └───────────────────────┘
```

**Отложенный пост:**
```
                          ┌───────────────────────┐
                          │ ↑ К посту             │
                          │ 📋 Заметки            │
                          │ ✏ Редактировать пост  │
                          │ 📢 Опубликовать        │
                          │ ✕ Отменить публикацию │
                          │ 🗑 Удалить             │
                          └───────────────────────┘
```

> В режиме заметок пункт [📋 Заметки] меняется на [💬 Чат].

| Пункт | Условие | Действие |
|---|---|---|
| ↑ К посту | всегда | Прокручивает ленту чата вверх к блоку поста |
| 📋 Заметки / 💬 Чат | всегда | Переключает режим пространства (чат ↔ заметки) |
| ✏ Редактировать пост | всегда | Прокручивает к блоку поста и открывает inline-редактирование |
| 📢 Опубликовать | черновик, отложен | Публикует пост немедленно |
| 🕐 Запланировать | черновик | Ставит пост в очередь на публикацию |
| ✕ Отменить публикацию | отложен | Переводит пост обратно в черновик |
| 🗑 Удалить | всегда | Удаляет пост с подтверждением; возвращает в ленту |

## Блок поста — строка статуса

Под текстом поста располагается строка в одну линию:

```
● Опубликован  👁 5100 · ❤ 87 · ↗ 23                [✏ Редактировать]
```

- Слева — статусный бейдж: `● Опубликован` / `◷ Отложено · дата` / `✏ Черновик`
- Рядом со статусом — метрики (только для опубликованных постов)
- Справа — кнопка `[✏ Редактировать]`, открывает inline-редактирование

## Правило формирования названия поста

| Начало текста | Название в шапке |
|---|---|
| `Два года я не мог нажать кнопку. Не потому что...` | `Два года я не мог нажать кнопку` |
| `Два года я не мог нажать кнопку\nНе потому что...` | `Два года я не мог нажать кнопку` |
| `Два года я не мог. Нажать кнопку. Вот почему...` | `Два года я не мог` |
| `Предложение первое.\nПредложение второе.\nТретье...` | `Предложение первое` (только первое предложение) |


## Элементы поля ввода чата

| Иконка | Действие |
|---|---|
| 📎 | Прикрепить медиа из поста |
| 💬 | Прикрепить файл с компьютера |
| ↑ | Отправить сообщение |
