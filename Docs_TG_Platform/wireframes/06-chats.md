## Шапка

- Заголовок `Чаты` слева.
- В центре: поиск по чатам и вкладки `Все` / `Глобальные` / `Локальные` (стартовая — `Все`).
- Справа: `← Назад` → `home`.

## Элементы

| # | Элемент | Описание |
|---|---|---|
| 1 | Вкладки `Все / Глобальные / Локальные` | В шапке, рядом с поиском |
| 2 | Карточка глобального чата | Название (первый запрос), превью последнего сообщения, дата |
| 3 | Карточка локального чата | Название локального чата (= первое сообщение), превью, дата, кнопка `→ пост` |
| 4 | `→ пост` | Только у локальных: переходит в пространство связанного поста |
| 5 | `🗑` | Удаляет чат (глобальный или конкретный локальный чат поста) |
| 6 | Клик на карточку | Глобальный → `gchat`, локальный → `post` с открытым этим локальным чатом |

## Вкладка «Все»

Показывает глобальные и локальные карточки в одном списке (глобальные сверху, затем локальные).

## Вкладка «Локальные»

Каждая карточка соответствует **одному локальному чату**, а не посту. У одного поста может быть несколько локальных чатов — несколько карточек с подписью связанного поста.
