## Шапка

- Заголовок `Чаты` слева.
- Поиск по чатам в центре шапки (фильтрует обе вкладки).

## Элементы

| # | Элемент | Описание |
|---|---|---|
| 1 | Переключатель `Глобальные / Локальные` | Под шапкой, две вкладки |
| 2 | Карточка глобального чата | Название (первый запрос), превью последнего сообщения, дата |
| 3 | Карточка локального чата | Название локального чата (= первое сообщение), привязанный пост подписью, превью, дата, кнопка `→ пост` |
| 4 | `→ пост` | Только у локальных: переходит в пространство связанного поста |
| 5 | `🗑` | Удаляет чат (для глобальных — глобальный чат; для локальных — конкретный локальный чат поста) |
| 6 | Клик на карточку | Глобальный → `gchat`, локальный → `post` с открытым именно этим локальным чатом (`currentPostChatId` подставляется напрямую) |

## Важно про вкладку «Локальные»

Каждая карточка во вкладке «Локальные» соответствует **одному локальному чату**, а не посту. У одного поста может быть произвольно много локальных чатов — в этом случае пост занимает несколько карточек, для контекста на каждой подписано название связанного поста.
