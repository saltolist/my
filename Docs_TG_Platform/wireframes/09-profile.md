## Элементы

### Вкладка "Канал"

| # | Элемент | Описание |
|---|---|---|
| 1 | [Изменить] у секции | Открывает поля редактирования inline |
| 2 | [+ Сегмент] | Добавляет новый сегмент ЦА |
| 3 | [+ Рубрика] | Создаёт новую рубрику |
| 4 | [🗑] у рубрики | Удаляет рубрику с подтверждением |
| 5 | [+ Добавить] референс | Добавляет ссылку на канал |

### Вкладка "Настройки"

| # | Элемент | Описание |
|---|---|---|
| 1 | Дропдаун модели | Выбор LLM-модели |
| 2 | Поле API-ключа | Masked input, кнопка Сохранить рядом |
| 3 | Поля параметров | Температура, max tokens |
| 4 | Редактор промпта | Textarea для кастомного системного промпта |
| 5 | [Сохранить всё] | Сохраняет все настройки ИИ разом |
| 6 | [Переподключить] | Перезапускает процесс подключения Telegram |

### Вкладка "Аналитика платформы"

| # | Элемент | Описание |
|---|---|---|
| 1 | Фильтр периода | 7 / 30 дней |
| 2 | Карточки использования ИИ | Запросы, токены, стоимость |
| 3 | Оценки ответов | Сводка лайков/дизлайков |
| 4 | [→] у агентского действия | Открывает детали: команда, результат |
| 5 | Активность | Сводные счётчики чатов, заметок, постов |
