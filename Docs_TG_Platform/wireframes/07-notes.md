# Заметки (`notes`)

Экран объединяет глобальные и локальные заметки.

## Структура

```
┌─────────────────────────────────────────────────────────────────┐
│  Заметки           [🔎 Поиск по заметкам]                       │
│ ─────────────────────────────────────────────────────────────── │
│  [Глобальные] [Локальные]                                       │
│  [Все] [● В контексте ИИ] [○ Не в контексте]   [+ Новая глоб.]  │
│ ─────────────────────────────────────────────────────────────── │
│  ┌────────────────────────┐  ┌────────────────────────┐         │
│  │ Заголовок              │  │ Заголовок              │         │
│  │ Превью тела...         │  │ Превью тела...         │         │
│  │ ● ИИ           [🗑]    │  │ 📌 Локальная → пост    │         │
│  └────────────────────────┘  └────────────────────────┘         │
└─────────────────────────────────────────────────────────────────┘
```

## Шапка

- Заголовок `Заметки` слева.
- Поиск по заголовку и тексту заметок в центре шапки.

## Строка фильтров

- Scope-переключатель `Глобальные / Локальные` слева.
- Фильтр `Все / В контексте ИИ / Не в контексте` рядом.
- Кнопка `+ Новая глобальная` в правом крае строки фильтров. Видна только в scope `Глобальные`.

## Карточка заметки

| Элемент | Описание |
|---|---|
| Заголовок | Название заметки |
| Превью | Фрагмент тела |
| `● ИИ / ○ ИИ` | Переключатель AI-флага (только у глобальных) |
| `🗑` | Удаление с подтверждением |
| `📌 Локальная · → пост` | Показывается только для локальной заметки |

## Действия

- Клик по карточке → открывает страницу заметки.
- Клик по `→ пост` → открывает пространство поста.
- Фильтр и поиск работают в реальном времени.
