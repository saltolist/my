## Обозначения на карточках

| Элемент | Описание |
|---|---|
| 🤖 | Заметка включена в контекст ИИ |
| [ ●] | Переключатель "В контексте ИИ" — активен (клик выключает) |
| [ ○] | Переключатель "В контексте ИИ" — неактивен (клик включает) |
| [🗑] | Удалить заметку (с подтверждением) |
| Клик на карточку | Переход на страницу заметки |

## Элементы страницы

| # | Элемент | Описание |
|---|---|---|
| 1 | Поиск | Фильтрует сетку в реальном времени по тексту заметки |
| 2 | Фильтр по контексту | Все / В контексте ИИ / Не в контексте |
| 3 | [+ Новая] | Создаёт глобальную заметку, открывает её страницу |
| 4 | Сетка карточек | По 2-3 карточки в ряд |
