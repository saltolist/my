# Заметки (`notes`)

Экран объединяет глобальные и локальные заметки.

## Структура

- Заголовок `Заметки`.
- Кнопка `+ Новая глобальная`.
- Поиск по заметкам.
- Scope-переключатель: `Глобальные / Локальные`.
- Фильтры: `Все / В контексте ИИ / Не в контексте`.
- Сетка карточек заметок.

## Карточка заметки

| Элемент | Описание |
|---|---|
| Заголовок | Название заметки |
| Превью | Фрагмент тела |
| `● ИИ / ○ ИИ` | Переключатель AI-флага |
| `🗑` | Удаление с подтверждением |
| `📌 Локальная • к посту` | Показывается только для локальной заметки |

## Действия

- Клик по карточке -> открывает страницу заметки.
- Клик по `к посту` -> открывает пространство поста.
- Фильтр и поиск работают в реальном времени.
