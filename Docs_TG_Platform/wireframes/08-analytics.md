# Аналитика канала (`analytics`)

## Состояние экрана

```
┌─────────────────────────────────────────────────────────────────┐
│ [7 дней] [30 дней] [90 дней] [Всё время]                        │
│                                                                 │
│ [Охваты] [Подписчики] [ER]                                      │
│                                                                 │
│ По рубрикам                                                     │
│ - Психология денег                                              │
│ - Разбор                                                        │
│ - Личный опыт                                                   │
│ - Новости                                                       │
│                                                                 │
│ Лучшие посты за период                                          │
│ 1. ...                                                   [→]    │
│ 2. ...                                                   [→]    │
└─────────────────────────────────────────────────────────────────┘
```

## Элементы

| Элемент | Действие |
|---|---|
| Табы периода | Переключение активного периода |
| Карточки метрик | Показывают значения и дельту |
| Блок `По рубрикам` | Сравнение рубрик в bar-формате |
| Таблица лучших постов | Переход к посту по `→` |

## Примечание

В демо нет отдельного дропдауна фильтра по рубрике и отдельного графика охватов по дням.
