## Элементы

| # | Элемент | Описание |
|---|---|---|
| 1 | Фильтр периода | Таб-переключатель: 7 / 30 / 90 дней / Всё время |
| 2 | Дропдаун рубрики | Фильтрует всю аналитику по рубрике |
| 3 | Сводные карточки | Охваты, подписчики, ER с динамикой (↑↓) за период |
| 4 | График охватов | Столбчатый или линейный по дням |
| 5 | Диаграмма по рубрикам | Горизонтальные бары с абсолютными значениями |
| 6 | Таблица лучших постов | Топ-3 по охватам с кнопкой [→] |
| 7 | [→] у поста | Открывает пространство этого поста |
