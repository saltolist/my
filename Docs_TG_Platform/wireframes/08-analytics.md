# Аналитика канала (`analytics`)

## Состояние экрана

```
┌─────────────────────────────────────────────────────────────────┐
│  Аналитика канала                                               │
│ ─────────────────────────────────────────────────────────────── │
│ [7 дней] [30 дней] [90 дней] [Всё время]                        │
│                                                                 │
│ [Охваты]   [Подписчики]   [ER]                                  │
│                                                                 │
│ По рубрикам                                                     │
│  Психология денег   ████████████████████ 6 200                  │
│  Разбор             ███████████████      4 800                  │
│  Личный опыт        ████████████         3 900                  │
│  Новости            ██████               2 100                  │
│                                                                 │
│ Лучшие посты за период                                          │
│  1. Синдром чистого листа с деньгами  5 100              [→]    │
│  2. Почему ИИС — не страшно           4 200              [→]    │
│  3. Личный опыт с ETF                 3 800              [→]    │
└─────────────────────────────────────────────────────────────────┘
```

## Шапка

- Заголовок `Аналитика канала` (`PageHeader`).
- Поиска и `← Назад` нет (верхнеуровневый экран).

## Элементы

| Элемент | Действие |
|---|---|
| Табы периода | Переключение активного периода (визуальное) |
| Карточки метрик | Показывают значения и дельту |
| Блок `По рубрикам` | Сравнение рубрик в bar-формате |
| Таблица `Лучшие посты` | Переход к посту по `→` |

## Примечание

В демо нет отдельного дропдауна фильтра по рубрике и отдельного графика охватов по дням. Данные — статичные mock-значения.
