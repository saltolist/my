## Шапка

- Заголовок `Лента` слева.
- Поиск по постам в центре шапки, выровнен по центру колонки карточек (`PageHeader`).
- Справа пусто (на верхнеуровневом экране нет `← Назад`).

## Структура ленты

Лента разделена на три фиксированные секции сверху вниз:

| Секция | Порядок внутри |
|---|---|
| **Опубликованные** | По дате публикации, новые сверху |
| **Отложенные** | По дате запланированной публикации, ближайшие сверху |
| **Черновики** | Порядок задаётся пользователем через drag-and-drop |

Фильтров по статусу нет — все три секции всегда видны. Заголовки секций центрированы над колонкой постов.

## Drag-and-drop черновиков

- Черновики можно перетаскивать и менять местами внутри секции.
- Иконка `⠿` слева на карточке — визуальный маркер drag-handle.
- Порядок сохраняется в памяти вкладки (до перезагрузки).

## Composer внизу ленты

- Авто-ресайз textarea вверх до 16 строк, дальше скролл.
- Кнопка `+` сразу открывает file picker — добавляет медиа в черновик. К одному черновику можно прикрепить несколько медиа; они показываются над textarea TG-стилем (см. `PostMediaBlock`) и Backspace’ом удаляются по одному.
- Селекторы LLM/Web Search в composer ленты не показываются — composer работает в режиме `feed`, без отправки в ИИ.

## Элементы

| # | Элемент | Описание |
|---|---|---|
| 1 | [✦ лого] вверху навигации | Клик переходит в `home`; на десктопе — сворачивание панели |
| 2 | `Глобальный чат` в навигации | Переход в `home` |
| 3 | Поиск в шапке | Фильтрация постов по тексту |
| 4 | Медиа в карточке | TG-стиль медиа-блок над текстом |
| 5 | Текст поста | Полный текст без обрезки |
| 6 | Метрики | Только для опубликованных: пилюли реакций, `👁` просмотры, `↗` репосты, `💬` комментарии (клик → режим `comments` поста) |
| 7 | Статус, дата, рубрика | Нижняя строка карточки |
| 8 | Поле ввода внизу | Создание нового черновика |
| 9 | `+` | Добавить медиа к черновику (открывает file picker) |
| 10 | `↑` | Создаёт черновик в секции `Черновики` |
| 11 | Клик на карточку | Открывает пространство поста |

## Название поста в хлебных крошках

Карточки ленты не имеют отдельного заголовка — отображается полный текст. Название поста вычисляется автоматически из текста и используется только в хлебных крошках пространства поста.

Правило: первое предложение первой строки до точки `.`; если точки нет — вся первая строка до переноса `\n`. Подробнее — в `03-post.md`.

## Рубрика на карточке

Рубрика отображается только у постов, созданных или размеченных через платформу. У постов, импортированных из Telegram без разметки, рубрика не отображается.

## Иконки статусов

| Иконка | Статус |
|---|---|
| ● | Опубликован |
| ◷ | Отложено (с датой и временем) |
| ✏ | Черновик |
| ⠿ | Drag-handle (только у черновиков) |
