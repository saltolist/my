## Структура ленты

Лента разделена на три фиксированные секции сверху вниз:

| Секция | Порядок внутри |
|---|---|
| **Опубликованные** | По дате публикации, новые сверху |
| **Отложенные** | По дате запланированной публикации, ближайшие сверху |
| **Черновики** | Порядок задаётся пользователем через drag-and-drop |

Фильтров по статусу нет — все три секции всегда видны.

## Drag-and-drop черновиков

- Черновики можно перетаскивать и менять местами внутри секции
- Иконка ⠿ слева на карточке — визуальный маркер drag-handle
- Порядок сохраняется

## Элементы

| # | Элемент | Описание |
|---|---|---|
| 1 | [лого] вверху навигации | Клик переходит в глобальный чат (главный экран) |
| 2 | [+ чат] в навигации | Создаёт новый глобальный чат |
| 3 | Текст поста в карточке | Полный текст поста без обрезки |
| 4 | Медиа в карточке | Изображения/видео поста отображаются в карточке под текстом |
| 5 | Метрики в карточке | Только для опубликованных постов, в нижней строке карточки |
| 6 | Статус, дата, рубрика | Нижняя строка карточки под текстом и медиа |
| 7 | Поле ввода внизу ленты | Ввод текста для нового поста-черновика |
| 8 | [📎] в поле ввода | Добавить медиа к новому черновику |
| 9 | [↑] в поле ввода | Отправляет текст, создаёт новый черновик в секции "Черновики" |
| 10 | Клик на карточку | Открывает пространство поста |

## Название поста в хлебных крошках

Карточки ленты не имеют отдельного заголовка — отображается полный текст. Название поста вычисляется автоматически из текста и используется только в хлебных крошках пространства поста.

Правило: первое предложение первой строки до точки `.`; если точки нет — вся первая строка до переноса `\n`. Подробнее — в `03-post.md`.

## Рубрика на карточке

Рубрика отображается только у постов, созданных или размеченных через платформу. У постов, импортированных из Telegram без разметки, рубрика не отображается.

## Иконки статусов

| Иконка | Статус |
|---|---|
| ● | Опубликован |
| ◷ | Отложено (с датой и временем) |
| ✏ | Черновик |
| ⠿ | Drag-handle (только у черновиков) |
