# Home (новый глобальный чат)

> Отдельной страницы "Главная" нет. Стартовый экран — `home`. Открывается кликом по логотипу или кнопке `Новый чат` в сайдбаре.

## Состояние экрана

```
┌─────────────────────────────────────────────────────────────────┐
│ NAV                             HOME                             │
│ ┌────────────┐ ┌──────────────────────────────────────────────┐ │
│ │ [лого]     │ │                                              │ │
│ │ [Новый чат]│ │                Чем помочь сегодня?           │ │
│ │            │ │                                              │ │
│ │ Лента      │ │  ┌────────────────────────────────────────┐  │ │
│ │ Чаты       │ │  │ Написать сообщение...                  │  │ │
│ │ Заметки    │ │  │                                        │  │ │
│ │ Аналитика  │ │  │ [+] [🧠 GPT-4.1 ▾] [🔎 Web ▾]    [↑]  │  │ │
│ │            │ │  └────────────────────────────────────────┘  │ │
│ │ Профиль    │ │                                              │ │
│ └────────────┘ └──────────────────────────────────────────────┘ │
└─────────────────────────────────────────────────────────────────┘
```

## Элементы

| Элемент | Действие |
|---|---|
| Логотип | Переход в `home` |
| `Новый чат` | Переход в `home` |
| Поле ввода | Текст нового глобального запроса; авто-ресайз вверх до 10 строк, дальше — скролл |
| `↑` | Создаёт новый глобальный чат и открывает `gchat` |
| `+` | Универсальное attach-меню (раскрывается вниз): `Прикрепить пост` (с подменю списка постов ленты), `Прикрепить файл с компьютера`, `Медиа из прикреплённых постов` (показывается, когда к сообщению уже привязан пост с медиа) |
| `🧠 …` | `ModelPicker` LLM-модели; dropdown с провайдерами и моделями (раскрывается вниз) |
| `🔎 …` | `ModelPicker` Web Search модели; рядом появляется флаг `Мультиответ`, если в профиле включено `Включить в мультиответ` у нескольких моделей |

## Прикрепления

- Выбранный пост или файл отображается «чипом» над textarea; чип можно удалить крестиком или Backspace’ом при пустом поле.
- Поддерживается синтаксис `@<title>;` в тексте — он автоматически превращается в чип-пост.
- Медиа в подменю — миниатюры 72×72, сетка адаптивная: 1–3 в ряд, 4 — сеткой 2×2, 5+ — три колонки со скроллом.
