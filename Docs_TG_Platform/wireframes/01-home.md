# Home (новый глобальный чат)

> Отдельной страницы "Главная" нет. Стартовый экран — `home`. Открывается кликом по логотипу или кнопке `Новый чат` в сайдбаре.

## Состояние экрана

```
┌─────────────────────────────────────────────────────────────────┐
│ NAV                             HOME                             │
│ ┌────────────┐ ┌──────────────────────────────────────────────┐ │
│ │ [лого]     │ │                                              │ │
│ │ [Новый чат]│ │                Чем помочь сегодня?           │ │
│ │            │ │                                              │ │
│ │ Лента      │ │  [📌] [📎]  Написать сообщение...     [↑]   │ │
│ │ Чаты       │ │                                              │ │
│ │ Заметки    │ │                                              │ │
│ │ Аналитика  │ │                                              │ │
│ │            │ │                                              │ │
│ │ Профиль    │ │                                              │ │
│ └────────────┘ └──────────────────────────────────────────────┘ │
└─────────────────────────────────────────────────────────────────┘
```

## Элементы

| Элемент | Действие |
|---|---|
| Логотип | Переход в `home` |
| `Новый чат` | Переход в `home` |
| Поле ввода | Текст нового глобального запроса |
| `↑` | Создаёт новый глобальный чат и открывает `gchat` |
| `📎` | Меню вложений (медиа из запиненных постов / файл с компьютера) |
| `📌` | Иконка закрепления (в демо без полного UI-цикла pin/unpin) |
