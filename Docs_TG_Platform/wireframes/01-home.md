# Глобальный чат (стартовый экран)

> Страница "Главная" как отдельного пункта меню нет. Стартовый экран — это глобальный чат. Попасть на него можно двумя способами: кликнуть на логотип вверху навигационной панели или нажать кнопку [+ Чат] в навигации.

## Состояние A — Новый чат (пустой)

```
┌─────────────────────────────────────────────────────────────────┐
│  NAV                  ГЛОБАЛЬНЫЙ ЧАТ                            │
│ ┌───────┐ ┌───────────────────────────────────────────────────┐ │
│ │[лого] │ │                                                   │ │
│ │       │ │                                                   │ │
│ │[+чат] │ │                                                   │ │
│ │       │ │                                                   │ │
│ │ Лента │ │            Чем могу помочь?                       │ │
│ │ Чаты  │ │                                                   │ │
│ │ Замет.│ │   ┌─────────────────────────────────────────┐    │ │
│ │ Аналит│ │   │ [📌][📎]  Написать сообщение...     [↑]   │   │ │
│ │       │ │   └─────────────────────────────────────────┘    │ │
│ │       │ │                                                   │ │
│ │ ───── │ │                                                   │ │
│ │[Профиль]│                                                   │ │
│ └───────┘ └───────────────────────────────────────────────────┘ │
└─────────────────────────────────────────────────────────────────┘
```

## Навигационная панель — полная структура

```
┌───────┐
│[лого] │  ← клик → глобальный чат (этот экран)
│       │
│[+чат] │  ← создать новый глобальный чат
│       │
│ Лента │
│ Чаты  │
│ Замет.│
│ Аналит│
│       │
│ ───── │
│       │
│ ───── │
│[Проф.]│
└───────┘
```

## Элементы


| #   | Элемент                      | Описание                                                  |
| --- | ---------------------------- | --------------------------------------------------------- |
| 1   | [лого]                       | Клик переходит на этот экран из любого места              |
| 2   | [+ чат]                      | Создаёт новый глобальный чат, сбрасывает текущий в "Чаты" |
| 3   | Заголовок "Чем могу помочь?" | Статичный текст, только при пустом чате                   |
| 4   | Поле ввода                   | Начало нового глобального чата                            |
| 5   | [📌]                         | Открывает выбор постов для закрепления в контексте        |
| 6   | [📎]                         | Прикрепить файл с компьютера                              |
| 7   | [↑]                          | Отправить сообщение                                       |
