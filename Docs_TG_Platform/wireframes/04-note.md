# Страница заметки (`note`)

Заметка поддерживает два режима: `view` и `edit`.

## Хлебные крошки

- Глобальная: `Заметки / Название`.
- Локальная: `Лента / Пост`.

Справа всегда: `← Назад` и меню `•••` с действием `🗑 Удалить заметку`.

## Режим `view`

Показывает:

- заголовок;
- `Ред.` и `Сохранить`;
- отрендеренный контент заметки;
- вложения;
- дату создания/изменения.

Дополнительно:

- для глобальной заметки: `● Учитывать ИИ / ○ Учитывать ИИ`;
- для локальной заметки: метка `📌 Локальная` и ссылка `→ пост`.

## Режим `edit`

Показывает:

- поле заголовка;
- textarea тела заметки;
- `📎 Файл`, `Просмотр`, `Сохранить`;
- те же служебные метки/даты.

## Поведение переходов

- при уходе со страницы с несохранёнными изменениями показывается предупреждение;
- `Сохранить` доступна в обоих режимах.
