# Страница заметки (`note`)

Заметка поддерживает два режима: `view` и `edit`.

## Шапка

```
┌─────────────────────────────────────────────────────────────────┐
│ Заметки / Название заметки                  [← Назад] [•••]     │
└─────────────────────────────────────────────────────────────────┘
```

- Слева — хлебные крошки: для глобальной `Заметки / Название`, для локальной `Лента / Пост / Название`.
- Справа: `← Назад`, меню `•••` с действием `🗑 Удалить заметку`.
- Заголовок в крошках укорачивается; сам заголовок заметки в теле страницы авто-подгоняется по ширине (`useFitTitleSize`).

## Режим `view`

Показывает:

- заголовок заметки;
- кнопки `Ред.` и `Сохранить` (в режиме view `Сохранить` неактивна);
- для глобальной заметки — переключатель `● Учитывать ИИ / ○ Учитывать ИИ` слева от кнопок;
- для локальной заметки — метка `📌 Локальная` и ссылка `→ пост`;
- отрендеренный текст заметки;
- блок `Вложения` (если есть файлы);
- дату создания/изменения внизу.

## Режим `edit`

Показывает:

- textarea заголовка (авто-подгонка шрифта под одну строку);
- кнопки `📎 Файл`, `Просмотр`, `Сохранить` (`Сохранить` активна только при изменениях);
- для локальной заметки — `📌 Локальная` + `→ пост`;
- textarea тела заметки с авто-ресайзом по контенту;
- блок `Вложения`.

## Поведение переходов

- При уходе со страницы с несохранёнными изменениями показывается подтверждение (`window.beforeunload`).
- `Сохранить` доступна только в режиме `edit` и активируется при изменениях.
