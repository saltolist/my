# Страница заметки

Заметка имеет два режима: **просмотр** (рендеренный Markdown) и **редактирование** (raw Markdown в textarea). Кнопка **[Сохранить]** присутствует в обоих режимах — сохранение не привязано к переходу между режимами.

---

## Хлебные крошки

```
Глобальная заметка:  Заметки / Название заметки...  [← Назад]  [•••]
Локальная заметка:   Лента / Пост... / Заметка...   [← Назад]  [•••]
```

## Меню [•••] заметки

```
                          ┌───────────────────┐
                          │ 🗑 Удалить заметку │
                          └───────────────────┘
```

| Пункт | Действие |
|---|---|
| 🗑 Удалить заметку | Удаляет заметку с подтверждением; возвращает на предыдущий экран |

## Элементы

| # | Элемент | Режим | Тип | Описание |
|---|---|---|---|---|
| 1 | [← Назад] | Оба | Оба | Возвращает на предыдущий экран без предупреждений |
| 2 | [Ред.] | Просмотр | Оба | Кнопка справа от названия, переключает в режим редактирования (raw Markdown) |
| 3 | [Просмотр] | Редактирование | Оба | Кнопка справа от названия, переключает в режим просмотра (рендеренный Markdown) |
| 4 | [● Учитывать ИИ] | Оба | Глобальные | Переключатель включения заметки в контекст ИИ; ● — активен, ○ — неактивен |
| 5 | [Сохранить] | **Оба** | Оба | Кнопка справа от названия, сохраняет заметку не меняя текущий режим |
| 6 | Ссылка на пост | Оба | Локальные | Клик → пространство поста |
| 7 | Заголовок | Просмотр | Оба | Статичный текст |
| 8 | Заголовок | Редактирование | Оба | Однострочное поле ввода |
| 9 | Тело | Просмотр | Оба | Отрендеренный Markdown |
| 10 | Тело | Редактирование | Оба | Raw Markdown в textarea |

## Поведение при переходах

- **[Ред.]** и **[Просмотр]** — переключают режим мгновенно, без предупреждений
- **[← Назад]** — возвращает на предыдущий экран без предупреждений
- **[Сохранить]** — сохраняет изменения заголовка и тела, режим не меняется
