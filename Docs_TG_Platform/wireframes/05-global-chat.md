# Глобальный чат (`gchat`)

## Состояние экрана

```
┌─────────────────────────────────────────────────────────────────┐
│ Чаты / Название чата                        [← Назад] [•••]     │
├─────────────────────────────────────────────────────────────────┤
│ [Я] ...                                                         │
│ [ИИ] ...                                                        │
│                                                                 │
│ ┌─────────────────────────────────────────────────────────────┐ │
│ │ Написать сообщение...                                       │ │
│ │ [+] [🧠 GPT-4.1 ▾] [🔎 Web ▾]                          [↑]  │ │
│ └─────────────────────────────────────────────────────────────┘ │
└─────────────────────────────────────────────────────────────────┘
```

## Шапка

- Слева — хлебные крошки `Чаты / Название чата`.
- Справа: `← Назад`, меню `•••` (`🗑 Удалить чат`).

## Сообщения

- ответы ИИ: одиночный или мультиответ; `AiMessageToolbar` (копирование, подсказка по моделям);
- пользовательские сообщения: редактирование по hover, ветки версий (`userBranches`), стрелки переключения;
- после правки — ответ-заглушка с пометкой «отредактировано».

## Composer

| Элемент | Действие |
|---|---|
| Textarea | Авто-ресайз вверх до 16 строк, дальше скролл |
| `+` | Attach-меню (раскрывается вверх): `Прикрепить пост`, `Прикрепить файл с компьютера`, `Медиа из прикреплённых постов` (показывается, когда к сообщению уже привязан хотя бы один пост с медиа). Подменю — миниатюры 72×72 с адаптивной сеткой и скроллом |
| `🧠 …` | LLM-`ModelPicker` (раскрывается вверх) |
| `🔎 …` | Web Search `ModelPicker`; рядом — флаг `Мультиответ`, если включено в профиле у нескольких моделей |
| `↑` | Отправка сообщения |

## Прикрепления

- Прикреплённый пост или файл отображается «чипом» над textarea; удаляется крестиком или Backspace’ом при пустом поле.
- Поддерживается синтаксис `@<title>;` в тексте — превращается в чип-пост.
- Сценарий «закрепить пост к чату как отдельную сущность» в текущем демо не реализован: вся привязка постов выполняется на уровне конкретного сообщения через composer.

## Примечание

В чате нет отдельного блока со списком закреплённых постов над полем ввода. Если у привязанных к сообщению постов есть медиа, оно становится доступным в пункте attach-меню `Медиа из прикреплённых постов`.
