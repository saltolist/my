## Хлебные крошки

```
Чаты / Название чата                       [← Назад]  [•••]
```

- Путь: **Чаты** → название чата (первый запрос или дата создания)
- [← Назад] справа — возвращает в раздел "Чаты"
- [•••] рядом с [← Назад] — кнопка контекстного меню
- При новом пустом чате: **Чаты / Новый чат**

## Меню [•••] глобального чата

```
                          ┌───────────────────┐
                          │ 🗑 Удалить чат     │
                          └───────────────────┘
```

| Пункт | Действие |
|---|---|
| 🗑 Удалить чат | Удаляет переписку с подтверждением; возвращает в список "Чаты" |

## Элементы поля ввода

| Иконка | Действие |
|---|---|
| 📌 | Открывает модальное окно выбора постов для закрепления |
| 📎 | Прикрепить медиа из запиненных постов |
| 💬 | Прикрепить файл с компьютера |
| ↑ | Отправить сообщение |

## Прочие элементы

| # | Элемент | Описание |
|---|---|---|
| 1 | [+ чат] в навигации | Сохраняет текущий чат в "Чаты", начинает новый |
| 2 | Карточка запиненного поста | Название поста, кнопка [✕] для открепления |
| 3 | [✕] на карточке | Убирает пост из контекста текущего чата |
