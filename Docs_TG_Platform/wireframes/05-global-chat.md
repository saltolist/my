# Глобальный чат (`gchat`)

## Состояние экрана

```
┌─────────────────────────────────────────────────────────────────┐
│ Чаты / Название чата                        [← Назад] [•••]     │
├─────────────────────────────────────────────────────────────────┤
│ [Я] ...                                                         │
│ [ИИ] ...                                                        │
│                                                                 │
│ [📌] [📎]  Написать сообщение...                          [↑]   │
└─────────────────────────────────────────────────────────────────┘
```

## Элементы

| Элемент | Действие |
|---|---|
| `← Назад` | Возврат в `Чаты` |
| `•••` | Контекстное меню |
| `🗑 Удалить чат` | Удаляет чат с подтверждением |
| `📌` | Иконка pin (без полного UI pin/unpin в демо) |
| `📎` | Меню вложений (медиа из запиненных постов / файл с компьютера) |
| `↑` | Отправка сообщения |

## Примечание

В текущем демо нет отдельного блока с карточками закреплённых постов над полем ввода.
