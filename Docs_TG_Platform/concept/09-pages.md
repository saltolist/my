# Структура страниц

Описание каждой страницы платформы: что отображается, какие действия доступны, какие кнопки активны.

---

## Навигация (постоянная левая панель)

Присутствует на всех страницах. Содержит:

- Логотип платформы вверху — клик переходит в глобальный чат
- Кнопка [+ Чат] — создаёт новый глобальный чат
- Пункты меню: Лента, Чаты, Заметки, Аналитика, Профиль
- Индикатор подключённого канала (название канала)

Отдельного пункта "Главная" в навигации нет. Роль стартового экрана выполняет глобальный чат.

---

## 1. Глобальный чат (стартовый экран)

**Назначение:** стартовый экран и основной интерфейс макро-ИИ. Открывается кликом на логотип или кнопкой [+ Чат].

**Что отображается:**
- При пустом чате: поле ввода по центру, chips быстрых запросов
- При активном чате: история переписки, запиненные посты (если есть), поле ввода внизу

**Хлебные крошки:** `Чаты / Название чата` — [← Назад] справа.

**Кнопки и действия:**

| Элемент | Действие |
|---|---|
| Поле ввода | Ввод сообщения макро-ИИ |
| Отправить (Enter / кнопка) | Отправляет сообщение |
| [+ Чат] в навигации | Сохраняет текущий чат в "Чаты", создаёт новый |
| [📌] Закрепить пост | Открывает выбор постов для закрепления в контексте |
| [📎] Медиа из постов | Медиа из запиненных постов |
| [💬] Файл с компьютера | Открывает файловый диалог |

---

## 2. Лента

**Назначение:** просмотр и управление всеми постами канала.

**Что отображается:**
- Три фиксированные секции сверху вниз: Опубликованные → Отложенные → Черновики
- Каждая карточка: название (первое предложение), статус, дата, рубрика (если добавлена через платформу), метрики (только у опубликованных)
- Кнопка [+ Новый пост] после последнего черновика

Фильтров нет. Все три секции всегда видны.

**Кнопки и действия:**

| Элемент | Действие |
|---|---|
| [+ Новый пост] | Создаёт черновик в конце секции черновиков, открывает его пространство |
| Drag-and-drop (⠿) | Перетаскивание черновиков для изменения порядка внутри секции |
| Клик на карточку | Открывает пространство поста |

---

## 3. Пространство поста

**Назначение:** работа с конкретным постом и его микро-ИИ.

**Хлебные крошки:** `Лента / Название поста` — [← Назад] справа.

### 3a. Шапка (постоянная)

**Что отображается:**
- Название поста — автоматически: первое предложение до точки или до переноса строки (если перед ним одно предложение; если уже два — берётся первое предложение)
- Статус поста (черновик / отложено / опубликовано)
- Метрики (охваты, реакции, репосты) — только для опубликованных

**Кнопки в шапке (режим чата):**

| Кнопка | Статус | Действие |
|---|---|---|
| [↑ К посту] | Все | Прокручивает чат к первому сообщению (тексту поста) |
| [📋 Заметки] | Все | Переключает область на сетку локальных заметок |
| [Опубликовать] | Черновик | Публикует в Telegram, статус → опубликован |
| [Отложить] | Черновик | Открывает пикер даты и времени |
| [Отменить] | Отложено | Отменяет отложенную публикацию, статус → черновик |

**Кнопки в шапке (режим заметок):**

| Кнопка | Статус | Действие |
|---|---|---|
| [↑ К посту] | Все | Прокручивает чат к первому сообщению |
| [►Чат] | Все | Переключает обратно на чат |
| [+ Заметка] | Все | Создаёт новую локальную заметку, переходит на её страницу |
| [Опубликовать] | Черновик | Публикует в Telegram |
| [Отложить] | Черновик | Открывает пикер даты и времени |
| [Отменить] | Отложено | Отменяет отложенную публикацию |

### 3b. Активная область — Чат (по умолчанию)

**Что отображается:**
- Первое сообщение — полный текст поста с кнопкой [✏ Редактировать]
- История переписки с микро-ИИ
- Поле ввода внизу

**Редактирование поста** — inline: кнопка [✏ Редактировать] под текстом поста превращает блок поста прямо на месте в редактируемую textarea. Переход на другой экран не происходит.

**Кнопки и действия:**

| Элемент | Действие |
|---|---|
| [✏ Редактировать] под постом | Блок поста становится textarea inline |
| [Отмена] в блоке редактирования | Откат изменений, блок возвращается в режим просмотра |
| [Сохранить] в блоке редактирования | Обновляет пост в БД, блок возвращается в режим просмотра |
| Поле ввода | Отправка сообщения микро-ИИ |
| Отправить (Enter / кнопка) | Отправляет сообщение, получает ответ ИИ |
| [📎] Медиа из поста | Прикрепляет медиа поста в чат |
| [💬] Файл с компьютера | Открывает файловый диалог |

### 3c. Активная область — Заметки поста

**Что отображается:**
- Сетка карточек локальных заметок, привязанных к этому посту

**Кнопки и действия:**

| Элемент | Действие |
|---|---|
| [+ Заметка] в шапке | Создаёт новую локальную заметку, открывает её страницу |
| Клик на карточку заметки | Переходит на страницу заметки |
| [🗑] на карточке | Удаляет заметку (с подтверждением) |
| [📋 Заметки] в шапке (повторный клик) | Возвращает активную область обратно к чату |

---

## 4. Страница заметки

**Назначение:** просмотр и редактирование заметки — как локальной, так и глобальной. Имеет два режима: просмотр и редактирование (как в Obsidian).

**Хлебные крошки:**
- Глобальная: `Заметки / Название заметки` — [← Назад] справа
- Локальная: `Лента / Название поста / Название заметки` — [← Назад] справа

### Режим просмотра (по умолчанию)

**Что отображается:**
- Заголовок заметки
- Тип заметки с переключателем/ссылкой и кнопка [Редактировать] на одной строке
- Отрендеренный Markdown
- Дата создания и изменения

**Кнопки:**

| Элемент | Действие |
|---|---|
| [← Назад] (в хлебных крошках) | Возвращает на предыдущий экран |
| [Редактировать] | Переключает в режим редактирования |
| [🗑] (в хлебных крошках) | Удаляет заметку с подтверждением |
| [● В контексте ИИ] / [○] | Только глобальные: переключатель рядом с кнопкой режима |
| Ссылка на пост | Только локальные: переходит в пространство поста |

### Режим редактирования

**Что отображается:**
- Поле заголовка
- Тип заметки с переключателем и кнопка [Просмотр] на одной строке
- Raw Markdown в textarea
- Кнопка [Сохранить] внизу

**Кнопки:**

| Элемент | Действие |
|---|---|
| [← Назад] | Предупреждение при несохранённых изменениях |
| [Просмотр] | Переключает в режим просмотра (с предупреждением если есть несохранённое) |
| [● В контексте ИИ] / [○] | Только глобальные: рядом с кнопкой [Просмотр] |
| [Сохранить] | Сохраняет изменения, остаётся в режиме редактирования |

---

## 5. Глобальный чат

**Назначение:** работа с макро-ИИ на уровне всего канала.

**Что отображается:**
- История переписки с макро-ИИ
- Список запиненных постов (если есть) — над полем ввода
- Поле ввода внизу

**Кнопки и действия:**

| Элемент | Действие |
|---|---|
| Поле ввода | Ввод сообщения макро-ИИ |
| Отправить (Enter / кнопка) | Отправляет сообщение, получает ответ ИИ |
| Кнопка "Закрепить пост" | Открывает модальное окно выбора постов для прикрепления в контекст |
| Клик на запиненный пост | Показывает превью поста с кнопкой открепить |
| Открепить пост | Убирает пост из контекста текущего чата |
| Прикрепить медиа с компьютера | Открывает файловый диалог |
| Прикрепить медиа из постов | Открывает список медиа из запиненных постов для выбора |
| Кнопка "Новый чат" | Создаёт новый глобальный чат, сохраняет текущий в разделе "Чаты" |

---

## 6. Чаты

**Назначение:** доступ ко всем сохранённым переписками.

**Что отображается:**
- Два раздела: "Локальные" и "Глобальные"
- Список чатов с названием (для локальных — название поста, для глобальных — первое сообщение или дата) и датой последнего сообщения

**Кнопки и действия:**

| Элемент | Действие |
|---|---|
| Клик на чат | Открывает переписку для продолжения |
| Кнопка "Перейти к посту" (у локального) | Открывает пространство связанного поста |
| Удалить чат | Удаляет переписку (с подтверждением) |

---

## 7. Заметки

**Назначение:** управление глобальными заметками.

**Что отображается:**
- Сетка карточек глобальных заметок
- Фильтр: все / в контексте ИИ / не в контексте
- Поиск по тексту

**Кнопки и действия:**

| Элемент | Действие |
|---|---|
| Кнопка "Новая заметка" | Создаёт новую глобальную заметку, открывает её страницу |
| Клик на карточку | Переходит на страницу заметки |
| Переключатель "В контексте ИИ" (на карточке) | Включает / выключает флаг без открытия страницы |
| Удалить заметку (на карточке) | Удаляет заметку (с подтверждением) |
| Поиск | Фильтрует заметки по тексту в реальном времени |

---

## 8. Аналитика канала

**Назначение:** метрики и статистика Telegram-канала.

**Что отображается:**
- Сводные показатели вверху (охваты, подписчики, ER)
- Графики по выбранному периоду
- Таблица лучших постов
- Разбивка по рубрикам

**Кнопки и действия:**

| Элемент | Действие |
|---|---|
| Выбор периода (7 / 30 / 90 дней / всё время) | Обновляет все графики и таблицы |
| Фильтр по рубрике | Фильтрует статистику по выбранной рубрике |
| Клик на пост в таблице | Открывает пространство этого поста |

---

## 9. Профиль

**Назначение:** контекст канала для ИИ и технические настройки. Разделён на три вкладки.

### Вкладка "Канал"

**Что отображается:**
- Секции: Целевая аудитория, Рубрики, Тон и голос, Структурные паттерны, Запрещённые темы, Бренд-бук, Конкуренты и референсы

**Кнопки и действия:**

| Элемент | Действие |
|---|---|
| Редактировать секцию | Открывает поля для редактирования |
| Сохранить | Сохраняет изменения, обновляет контекст ИИ |
| Добавить рубрику | Создаёт новую рубрику с формой настройки |
| Удалить рубрику | Удаляет рубрику (с подтверждением) |
| Добавить сегмент ЦА | Добавляет новый сегмент аудитории |
| Добавить референс | Добавляет ссылку на канал-референс или конкурента |

### Вкладка "Настройки"

**Что отображается:**
- Блок: ИИ-движок (модель, API-ключ, системный промпт, параметры)
- Блок: Telegram (параметры подключения)

**Кнопки и действия:**

| Элемент | Действие |
|---|---|
| Выбрать модель (dropdown) | Меняет активную модель |
| Поле API-ключа + Сохранить | Обновляет ключ |
| Редактировать системный промпт | Открывает текстовое поле для кастомизации |
| Сохранить параметры | Сохраняет температуру, max tokens и др. |
| Переподключить Telegram | Перезапускает процесс подключения к каналу |

### Вкладка "Аналитика платформы"

**Назначение:** метрики использования самой платформы — ИИ и активность пользователя.

**Что отображается:**
- Блок: Использование ИИ (токены, запросы, стоимость, оценки ответов)
- Блок: История агентских действий (что было сделано, когда, результат)
- Блок: Активность на платформе (чаты, заметки, посты по периодам)

**Кнопки и действия:**

| Элемент | Действие |
|---|---|
| Выбор периода (7 / 30 дней) | Обновляет все блоки статистики |
| Клик на агентское действие | Показывает детали: команда, что изменено, результат |
| Оценить ответ (лайк / дизлайк в истории) | Обновляет оценку конкретного ответа ИИ |
