# Суть продукта

## Проблема

Авторы Telegram-каналов работают в разрозненных инструментах. Идеи хранятся в заметках, план публикаций — в таблице, сам постинг идёт через отдельный бот. ИИ-инструменты не знают контекст канала, аудитории и истории постов, поэтому их помощь поверхностная.

Нет единого места, где всё связано: посты, планирование, ИИ, аналитика и база знаний о канале.

## Решение

TG Platform — единое рабочее пространство для автора канала. Ядро платформы:

- **Лента постов** — отражение реального канала + место для планирования
- **Контекстный ИИ** — знает всё о канале и работает на двух уровнях: пост и канал целиком
- **Профиль канала** — база знаний для ИИ: аудитория, рубрики, стиль, правила
- **Заметки** — свободное пространство для идей и исследований
- **Аналитика** — метрики канала и ИИ в одном месте

## Для кого

Платформа создана для одного пользователя — автора, который ведёт канал самостоятельно. Это личный инструмент, а не SaaS для команд.

## Принципы

### 1. Контекст — главный актив
Всё, что пользователь добавляет в профиль, заметки, рубрики — питает ИИ. Чем подробнее заполнен профиль, тем точнее и полезнее ответы ИИ. Качество помощи прямо пропорционально вложенному контексту.

### 2. Пост — живое пространство
Пост — это не просто опубликованный текст. Это место для работы: обсуждения с ИИ, итераций, связей с другими постами и заметками. После публикации пост не "умирает" — он остаётся активным пространством.

### 3. ИИ — соавтор, не замена
Автор принимает все решения. ИИ усиливает возможности: предлагает идеи, помогает с текстом, анализирует эффективность. Последнее слово всегда за человеком.

### 4. Open source, self-hosted
Полный открытый исходный код. Каждый разворачивает платформу у себя. Никакой привязки к облаку, никакой передачи данных третьим сторонам. Пользователь сам выбирает ИИ-движок.

## Позиционирование

TG Platform — не просто планировщик постов и не просто ИИ-чатбот. Это рабочая среда автора, где все инструменты связаны через общий контекст.

| Что это НЕ | Что это ЕСТЬ |
|---|---|
| Планировщик постов (без ИИ) | Рабочая среда с контекстным ИИ |
| Генератор контента | Соавтор, знающий канал |
| SaaS для команд | Личный self-hosted инструмент |
| Замена Telegram | Надстройка над Telegram-каналом |
