# ИИ-система

## Двухуровневая архитектура

Платформа использует два типа ИИ-помощников, работающих на разных уровнях.

---

## Микро-ИИ (уровень поста)

Каждое пространство поста имеет своего контекстного помощника. Он привязан к конкретному посту и знает всё о нём.

**Контекст микро-ИИ:**
- Содержание поста (текст, медиа, ссылки)
- Метрики поста (охваты, реакции, репосты)
- История редактирования
- Локальные заметки, прикреплённые к посту
- Рубрика поста и её правила из профиля
- Общие настройки канала из профиля

**Что умеет:**
- Доработать, переписать или расширить пост
- Предложить заголовки и подписи к медиа
- Создать тред или продолжение темы
- Объяснить, почему пост сработал или нет
- Предложить форматы для этой же темы

**Агентские действия микро-ИИ:**

Микро-ИИ может совершать действия по прямой команде пользователя в чате. Каждое действие — это запрос → действие → чёткий результат. Автономной работы без команды нет.

| Команда пользователя | Действие агента | Результат |
|---|---|---|
| "Отредактируй пост: убери последний абзац" | Редактирует текст поста | Пост обновлён, изменение видно в шапке |
| "Опубликуй этот пост" | Публикует черновик в канал | Статус поста меняется на "опубликован" |

---

## Макро-ИИ (уровень канала)

Глобальный помощник в отдельном чате. Видит весь канал целиком и работает как стратегический советник.

**Контекст макро-ИИ:**
- Все посты канала (или релевантная выборка через MemPalace)
- Профиль канала полностью
- Глобальные заметки (или помеченные флагом "в контекст")
- Метрики канала
- Запиненные посты — передаются с акцентом как приоритетный контекст

**Запин постов.**
Пользователь может добавить один или несколько конкретных постов в контекст текущего чата отдельной кнопкой. Запиненные посты учитываются в первую очередь — это позволяет фокусировать макро-ИИ на нужных материалах на фоне всего канала.

**Что умеет:**
- Составить контент-план на неделю/месяц
- Проанализировать, какие рубрики работают
- Дать рекомендации по стратегии роста
- Сравнить несколько постов и объяснить разницу в результатах
- Найти паттерны в успешных и неуспешных постах

**Агентские действия макро-ИИ:**

| Команда пользователя | Действие агента | Результат |
|---|---|---|
| "Создай черновик поста на тему X" | Создаёт новый пост-черновик | Черновик появляется в ленте |
| "Опубликуй пост [название]" | Публикует указанный черновик | Статус поста меняется на "опубликован" |

---

## Принцип работы агентов

Агенты работают только по прямой команде пользователя. Никакой автономной фоновой активности нет.

**Цикл:** команда пользователя → действие агента → чёткий видимый результат.

Пользователь всегда знает, что именно сделал агент, и может это проверить. История агентских действий хранится в технических настройках профиля.

---

## Медиа в чате

В чат (как микро-, так и макро-уровня) можно добавить медиафайлы:
- **Из поста** — прикрепить медиа, уже привязанное к посту (только в микро-чате)
- **С компьютера** — загрузить файл напрямую в чат

ИИ видит прикреплённые медиа и может работать с ними: описать, предложить подпись, проанализировать в контексте поста.

---

## Контекст ИИ

Качество ответов прямо зависит от заполненности профиля и заметок. Контекст собирается из нескольких источников:

| Источник | Что даёт ИИ | Уровень |
|---|---|---|
| Профиль канала | Тон, стиль, ЦА, рубрики, запреты | Оба |
| История постов | Паттерны, частота, что работало | Макро |
| Глобальные заметки (с флагом) | Идеи, исследования, база знаний | Макро |
| Локальные заметки поста | Заметки, привязанные к конкретному посту | Микро |
| Метрики канала | Какой контент получает охваты | Макро |
| Конкретный пост | Содержание и история поста | Микро |
| Запиненные посты | Приоритетный контекст по выбору пользователя | Макро |

---

## Настройка ИИ-движка

Платформа не привязана к конкретной модели. Пользователь выбирает в технических настройках:

- **Модель** — любая совместимая через OpenAI-совместимый API
- **API-ключ** — свой ключ, хранится локально
- **Системный промпт** — возможность дополнить базовый промпт
- **Параметры** — температура, максимальная длина ответа

Можно подключить OpenAI, Anthropic, Mistral, локальную модель через Ollama и любой другой совместимый провайдер.
