# Пространства платформы

Платформа организована как иерархия пространств. Каждое пространство имеет свой контекст и своего ИИ-помощника.

## Структура навигации

```
TG Platform
├── [лого] / [+ Чат]             ← стартовый экран = глобальный чат
├── Лента                        ← все посты канала
│   └── Пространство поста       ← подпространство каждого поста
│       └── Страница заметки     ← отдельная страница локальной заметки
├── Чаты                         ← все сохранённые чаты (локальные и глобальные)
├── Заметки                      ← глобальные заметки
│   └── Страница заметки         ← отдельная страница глобальной заметки
├── Аналитика канала             ← метрики Telegram-канала (в навигации)
└── Профиль
    ├── Канал                    ← контекст для ИИ
    ├── Настройки                ← ИИ-движок, Telegram
    └── Аналитика платформы      ← метрики ИИ и использования платформы
```

---

## Глобальный чат (стартовый экран)

Отдельного пункта "Главная" в навигации нет. Стартовый экран — это глобальный чат. Попасть на него можно двумя способами:
- Клик на **логотип** вверху навигационной панели
- Кнопка **[+ Чат]** в навигации (создаёт новый чат, сохраняя текущий)

Подробнее о глобальном чате — ниже в разделе "Глобальный чат (макро-ИИ)".

---

## Лента

Отдельная страница в левой навигационной панели. Отображает все посты канала в трёх фиксированных секциях сверху вниз: Опубликованные → Отложенные → Черновики. Фильтров нет.

**Статусы постов:**
- **Черновик** — пост создан, но не запланирован и не опубликован
- **Отложенная публикация** — указана дата и время, пост выйдет автоматически
- **Опубликован** — пост опубликован в Telegram-канале

**Что можно делать в ленте:**
- Просматривать все посты по всем статусам
- Создавать новые посты — кнопка [+ Новый пост] после последнего черновика
- Перетаскивать черновики для изменения порядка (drag-and-drop)
- Видеть базовые метрики в карточке поста (только у опубликованных)
- Переходить в пространство конкретного поста

**Рубрика** отображается только у постов, созданных или размеченных через платформу. Импортированные посты без разметки рубрику не показывают.

## Пространство поста

Открывается при нажатии на пост в ленте. Это чат-интерфейс, привязанный к конкретному посту.

### Шапка

Шапка всегда видна над активной областью. Содержит:

- **Название поста** — автоматически: первое предложение до точки или до переноса строки (если перед ним одно предложение; если уже два — берётся первое предложение по точке)
- **Статус** — черновик / отложена / опубликован
- **Метрики** — охваты, реакции, репосты (только для опубликованных)
- **[↑ К посту]** — прокручивает чат к первому сообщению (тексту поста)
- **[📋 Заметки]** — переключает область под шапкой на сетку локальных заметок
- **[+ Заметка]** — создаёт новую локальную заметку, открывает её страницу (доступна в обоих режимах)

### Первое сообщение — сам пост

Чат всегда начинается с сообщения, содержащего полный текст поста. Это сообщение особое:

- Оно всегда первое в чате и отражает актуальное состояние поста
- **Редактирование вручную** — пользователь может отредактировать текст прямо в этом сообщении. При сохранении пост обновляется в базе данных
- **Редактирование через ИИ** — когда микро-ИИ изменяет пост по команде, первое сообщение обновляется автоматически
- Изменения текста в сообщении и изменения поста всегда синхронизированы: одно зеркалит другое

Таким образом, пользователь может работать с постом либо напрямую (редактируя первое сообщение), либо через ИИ в чате — результат одинаков.

### Область под шапкой

Под шапкой находится одна из двух областей — переключаются кнопками в шапке:

**Чат с микро-ИИ (по умолчанию)**
Переписка с ИИ-помощником. ИИ может:
- Обсуждать пост — доработать, переписать, расширить, предложить продолжение
- Редактировать пост по команде (изменения отражаются в первом сообщении)
- Публиковать черновик или выставить отложенную публикацию по команде

**Сетка локальных заметок**
Отображается при нажатии на кнопку [📋 Заметки] в шапке. Показывает все заметки, привязанные к этому посту, в виде сетки. Создать новую заметку можно кнопкой [+ Заметка] в шапке — она доступна в обоих режимах.

### Медиа

В чат поста можно добавить медиафайлы двумя способами:
- Прикрепить из самого поста (медиа, уже привязанное к посту)
- Загрузить с компьютера напрямую в чат

### Контекст микро-ИИ

- Полный текст и медиа поста
- Метрики поста
- История редактирования
- Локальные заметки, прикреплённые к посту
- Рубрика и правила из профиля

## Глобальный чат (макро-ИИ)

Доступен через главную страницу (поле ввода) или как отдельный раздел. ИИ-помощник, работающий на уровне всего канала.

**Контекст:**
- Весь канал: посты, профиль, глобальные заметки, метрики
- Запиненные посты — передаются в контекст с акцентом как приоритетный материал

**Пины постов.**
Пользователь добавляет один или несколько постов в контекст текущего чата отдельной кнопкой. ИИ учитывает их в первую очередь.

**Медиа в глобальном чате.**
В чат можно прикрепить медиафайлы двумя способами:
- Загрузить с компьютера
- Прикрепить из запиненных постов (медиа доступно только из тех постов, которые запинены в текущем чате)

**Примеры запросов:**
- "Составь контент-план на следующую неделю"
- "Какие рубрики работают хуже всего?"
- "Проанализируй эти два поста [запинены] — почему один сработал, а другой нет?"
- "Предложи 5 тем для постов на основе моей аудитории"

## Чаты

Отдельная вкладка в навигации. Хранит все переписки в двух категориях:

- **Локальные чаты** — чаты из пространств конкретных постов (микро-уровень)
- **Глобальные чаты** — чаты из глобального чата (макро-уровень)

Позволяет вернуться к любой прошлой переписке и продолжить её.

## Профиль

Два блока в одном разделе:

### Профиль канала
Контекст для ИИ: портрет аудитории, рубрики, правила, стиль, бренд-бук. Подробнее — в [04-channel-profile.md](./04-channel-profile.md).

### Технические настройки
Доступны внутри профиля, не вынесены в основную навигацию:
- Выбор и настройка ИИ-движка (модель, API-ключ)
- Технические метрики ИИ (токены, запросы, стоимость)
- История агентских действий
- Настройки интеграции с Telegram
