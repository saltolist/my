# Модули платформы (актуально для demo)

Документ фиксирует модули в текущем демо. Это рабочая карта интерфейса, а не целевая спецификация будущих версий.

## Структура навигации

```
TG Platform
├── [лого] / [Новый чат]         ← стартовый экран (новый глобальный чат)
├── Лента                        ← все посты: опубликованные / отложенные / черновики
│   └── Пространство поста       ← чат поста + режим заметок поста
│       └── Страница заметки     ← локальная заметка
├── Чаты                         ← глобальные и локальные переписки
├── Заметки                      ← список глобальных и локальных заметок
│   └── Страница заметки         ← глобальная или локальная
├── Аналитика канала             ← сводка по каналу
└── Профиль канала
    ├── Канал                    ← базовый контекст
    ├── Настройки                ← ИИ-движок и Telegram
    └── Аналитика платформы      ← usage-метрики
```

---

## Глобальный чат

Глобальный чат состоит из двух экранов:

- `home` — новый чат (пустое состояние),
- `gchat` — уже созданный чат.

**Доступно в демо:**

- отправка сообщений;
- сохранение чатов в разделе `Чаты`;
- удаление чата из меню `•••`;
- прикрепление файла с компьютера;
- прикрепление медиа из уже запиненных постов.

**Ограничение демо:** UI-цикл закрепления/открепления постов в чате пока не реализован полностью.

---

## Лента

Лента показывает посты в фиксированном порядке секций:

1. `Опубликованные`
2. `Отложенные`
3. `Черновики`

**Что работает:**

- создание нового черновика через поле ввода внизу;
- переход в пространство поста по клику на карточку;
- отображение статуса, даты, рубрики и метрик (метрики только у опубликованных);
- drag-handle `⠿` у черновиков для drag-and-drop-перестановки в пределах секции.

**Замечание:** drag-and-drop в демо — это часть фронт-поведения (см. [12-demo.md](./12-demo.md)); порядок черновиков сохраняется в памяти вкладки и сбрасывается при перезагрузке.

---

## Пространство поста

Локальное пространство конкретного поста.

**Шапка:**

- хлебные крошки `Лента / Название`;
- кнопка `← Назад`;
- кнопка `↑ К посту` (показывается, когда первое сообщение ушло выше области видимости);
- меню `•••`.

**Меню `•••`:**

- переключение `📋 Заметки` / `💬 Чат`,
- публикация и планирование (для черновиков),
- отмена планирования (для отложенных),
- удаление поста.

**Режим `chat`:**

- первое сообщение = текст поста;
- inline-редактирование (`✏ Редактировать` + `Отмена/Сохранить`);
- история локального чата;
- прикрепление медиа из поста и файла с компьютера.

**Режим `notes`:**

- сетка локальных заметок поста;
- создание новой заметки;
- переключение AI-флага и удаление заметки.

---

## Чаты

Раздел содержит две вкладки:

- `Глобальные` — сохранённые глобальные чаты;
- `Локальные` — переписки из пространств постов.

Для локальных чатов доступна кнопка перехода `→ пост`.

---

## Заметки

Раздел заметок в демо объединяет два scope:

- `Глобальные`,
- `Локальные`.

Есть:

- поиск по заголовку и тексту;
- фильтр `Все / В контексте ИИ / Не в контексте`;
- переключение AI-флага на карточке;
- удаление;
- переход на страницу заметки.

---

## Аналитика канала

Доступна отдельным экраном в навигации.

В демо показывает:

- период (`7/30/90/Всё время`);
- сводные карточки (`Охваты`, `Подписчики`, `ER`);
- блок `По рубрикам`;
- таблицу `Лучшие посты за период` с переходом к посту.

---

## Профиль

Три вкладки:

- `Канал` — ЦА, рубрики, тон и голос, запрещённые темы;
- `Настройки` — модель, API-ключ, параметры, системный промпт, Telegram;
- `Аналитика платформы` — запросы, токены, стоимость, общая активность.
