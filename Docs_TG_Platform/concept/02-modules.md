# Модули платформы (актуально для demo)

Документ фиксирует модули в текущем демо. Это рабочая карта интерфейса, а не целевая спецификация будущих версий.

## Структура навигации

```
TG Platform
├── [лого] / [Новый чат]         ← стартовый экран (новый глобальный чат)
├── Лента                        ← все посты: опубликованные / отложенные / черновики
│   └── Пространство поста       ← чат поста + режим заметок поста
│       └── Страница заметки     ← локальная заметка
├── Чаты                         ← глобальные и локальные переписки
├── Заметки                      ← список глобальных и локальных заметок
│   └── Страница заметки         ← глобальная или локальная
├── Аналитика канала             ← сводка по каналу
└── Профиль канала
    ├── Канал                    ← базовый контекст
    ├── Настройки                ← ИИ-движок и Telegram
    └── Аналитика платформы      ← usage-метрики
```

---

## Глобальный чат

Глобальный чат состоит из двух экранов:

- `home` — новый чат (пустое состояние),
- `gchat` — уже созданный чат.

**Доступно в демо:**

- отправка сообщений;
- сохранение чатов в разделе `Чаты`;
- удаление чата из меню `•••`;
- composer с выбором LLM-модели, Web Search-модели и режимом мультиответа;
- attach-меню `+` со следующими действиями:
  - `Прикрепить пост` — выбор поста из ленты, привязывается к сообщению как «чип» (синтаксис `@<title>;` делает то же самое);
  - `Медиа из прикреплённых постов` — миниатюры медиа только из уже привязанных к сообщению постов;
  - `Прикрепить файл с компьютера`.

**Ограничение демо:** отдельной сущности «закреплённые посты чата» в UI больше нет — всё прикрепление выполняется на уровне конкретного сообщения через composer.

---

## Лента

Лента показывает посты в фиксированном порядке секций:

1. `Опубликованные`
2. `Отложенные`
3. `Черновики`

**Что работает:**

- создание нового черновика через поле ввода внизу (в composer’е ленты `+` сразу открывает выбор файла; к одному черновику можно прикрепить несколько медиа);
- поиск по постам в шапке;
- переход в пространство поста по клику на карточку;
- отображение статуса, даты, рубрики и метрик (метрики только у опубликованных), TG-стиль медиа-блок над текстом карточки;
- drag-handle `⠿` у черновиков для drag-and-drop-перестановки в пределах секции.

**Замечание:** drag-and-drop в демо — это часть фронт-поведения (см. [12-demo.md](./12-demo.md)); порядок черновиков сохраняется в памяти вкладки и сбрасывается при перезагрузке.

## Пространство поста

Локальное пространство конкретного поста.

**Шапка:**

- хлебные крошки `Лента / Название` слева;
- справа в одной группе: `↑ К посту` (показывается, когда первое сообщение ушло выше области видимости), `← Назад`, меню `•••`.

**Меню `•••`:**

- переключение `📋 Заметки` / `💬 Чат`,
- публикация и планирование (для черновиков),
- отмена планирования (для отложенных),
- удаление поста.

**Режим `chat`:**

- первое сообщение = карточка поста (TG-стиль медиа над текстом + текст);
- inline-редактирование (`✏ Редактировать` + `Отмена/Сохранить`): textarea авто-ресайзится от исходного размера, медиа остаётся сверху, есть кнопка `🖼 Добавить медиа`;
- история локального чата;
- composer внизу: `+` открывает меню с `Прикрепить пост`, `Медиа из поста`, `Прикрепить файл с компьютера` и `Медиа из прикреплённых постов` (когда привязаны другие посты).

**Режим `notes`:**

- сетка локальных заметок поста;
- создание новой заметки;
- переключение AI-флага и удаление заметки.

---

## Чаты

Раздел содержит две вкладки:

- `Глобальные` — сохранённые глобальные чаты;
- `Локальные` — переписки из пространств постов.

Для локальных чатов доступна кнопка перехода `→ пост`.

---

## Заметки

Раздел заметок в демо объединяет два scope:

- `Глобальные`,
- `Локальные`.

Есть:

- поиск по заголовку и тексту;
- фильтр `Все / В контексте ИИ / Не в контексте`;
- переключение AI-флага на карточке;
- удаление;
- переход на страницу заметки.

---

## Аналитика канала

Доступна отдельным экраном в навигации.

В демо показывает:

- период (`7/30/90/Всё время`);
- сводные карточки (`Охваты`, `Подписчики`, `ER`);
- блок `По рубрикам`;
- таблицу `Лучшие посты за период` с переходом к посту.

---

## Профиль

Три вкладки:

- `Канал` — ЦА, рубрики, тон и голос, запрещённые темы (в демо вкладка показана упрощённо, как статичный демо-профиль; полная модель — в [05-channel-profile.md](./05-channel-profile.md));
- `Настройки` — несколько LLM-моделей и несколько Web Search моделей с флагом «Включить в мультиответ», переключатель `Мультиответ`, системный промпт, MTProto-форма Telegram (api_id, api_hash, телефон, код, sessionName, syncMode, подключение канала). Провайдеры и режимы выбираются через тот же `ModelPicker`, что и в composer’е; поля API-ключей/промпта/Telegram оформлены с явными рамками. Параметры `температура` и `max tokens` пока не вынесены в UI;
- `Аналитика платформы` — запросы, токены, стоимость, общая активность.
