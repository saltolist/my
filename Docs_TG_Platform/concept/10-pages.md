# Структура страниц (актуально для demo)

Документ описывает страницы именно в текущем `demo/index.html`.

---

## Навигация (левая панель)

Постоянна на всех экранах.

- Логотип `TG Platform` -> переход в `home`.
- Кнопка `Новый чат` -> переход в `home`.
- Пункты: `Лента`, `Чаты`, `Заметки`, `Аналитика`, `Профиль`.

---

## 1. `home` (новый глобальный чат)

**Назначение:** старт создания нового глобального чата.

**Элементы:**

- заголовок `Чем помочь сегодня?`;
- поле ввода;
- кнопки `📌`, `📎`, `↑`.

**Поведение:**

- отправка сообщения создаёт новый глобальный чат и открывает экран `gchat`.
- `📎` открывает меню вложений (медиа из уже запиненных постов + файл с компьютера).

---

## 2. `gchat` (открытый глобальный чат)

**Назначение:** переписка с глобальным ИИ.

**Элементы:**

- крошки `Чаты / Название чата`;
- `← Назад`;
- меню `•••` с действием `Удалить чат`;
- история сообщений;
- поле ввода с `📌`, `📎`, `↑`.

**Замечание:** в текущем демо нет визуального блока списка закреплённых постов внутри тела чата.

---

## 3. `feed` (лента)

**Назначение:** просмотр и создание постов.

**Структура:**

- секции `Опубликованные`, `Отложенные`, `Черновики`;
- карточка поста: текст, опционально медиа, статус, дата, рубрика, метрики;
- нижнее поле ввода для нового черновика.

**Действия:**

- `↑` внизу создаёт черновик;
- клик на карточку открывает `post`.

**Замечание:** drag-handle `⠿` отображается только визуально; drag-and-drop не реализован.

---

## 4. `post` (пространство поста)

**Назначение:** локальный чат по посту + локальные заметки.

**Шапка:**

- крошки `Лента / Название`;
- `← Назад`;
- `↑ К посту` (появляется только когда первое сообщение вне видимости);
- меню `•••`.

**Меню `•••`:**

- `📋 Заметки` / `💬 Чат`;
- `📢 Опубликовать` (для черновика/отложенного);
- `🕐 Запланировать` (для черновика);
- `✕ Отменить публикацию` (для отложенного);
- `🗑 Удалить`.

**Режим чата:**

- первое сообщение содержит текст поста;
- inline-редактирование текста (`✏ Редактировать`);
- поле ввода с вложениями `📎`.

**Режим заметок:**

- сетка локальных заметок;
- создание, удаление, AI-флаг.

---

## 5. `note` (страница заметки)

**Назначение:** просмотр/редактирование глобальной или локальной заметки.

**Общее:**

- крошки, `← Назад`, меню `•••` (`Удалить заметку`);
- два режима: `view` и `edit`;
- кнопка `Сохранить` есть в обоих режимах.

**Глобальная заметка:**

- доступен переключатель `● Учитывать ИИ / ○ Учитывать ИИ`.

**Локальная заметка:**

- отображается метка `📌 Локальная` и ссылка `→ пост`.

**Переходы:**

- при несохранённых изменениях показывается предупреждение при уходе со страницы.

---

## 6. `chats` (список чатов)

**Назначение:** доступ к перепискам.

**Секции:**

- вкладки `Глобальные` и `Локальные`.

**Действия:**

- клик по глобальному чату -> `gchat`;
- клик по локальному чату -> `post`;
- `→ пост` у локального чата;
- `🗑` удаление глобального чата.

---

## 7. `notes` (список заметок)

**Назначение:** единый каталог заметок.

**Элементы:**

- кнопка `+ Новая глобальная`;
- поиск;
- scope-переключатель `Глобальные / Локальные`;
- фильтр `Все / В контексте ИИ / Не в контексте`;
- сетка карточек.

**Действия:**

- открыть заметку;
- переключить AI-флаг;
- удалить заметку;
- перейти к посту из локальной заметки.

---

## 8. `analytics` (аналитика канала)

**Назначение:** краткая статистика канала.

**Элементы:**

- вкладки периода `7/30/90/Всё время`;
- карточки `Охваты`, `Подписчики`, `ER`;
- блок `По рубрикам` (bar-список);
- таблица `Лучшие посты за период`.

**Действия:**

- выбор периода (визуальное переключение таба);
- переход к посту из таблицы.

---

## 9. `profile` (профиль канала)

Три вкладки:

### `Канал`

- Целевая аудитория;
- Рубрики;
- Тон и голос;
- Запрещённые темы.

### `Настройки`

- ИИ-движок: модель, API-ключ, температура, max tokens, системный промпт;
- Telegram: подключённый канал и кнопка переподключения.

### `Аналитика платформы`

- Использование ИИ: запросы, токены, стоимость;
- Активность: чаты, заметки, посты.
