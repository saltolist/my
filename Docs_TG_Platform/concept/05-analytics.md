# Аналитика

Аналитика разделена на два блока с разными назначениями и разным расположением в интерфейсе.

---

## Аналитика канала

Доступна из основной навигации. Помогает понять, как развивается канал и какой контент работает.

### Охваты
- Просмотры по каждому посту
- Динамика охватов по времени (неделя / месяц / всё время)
- Пики активности аудитории

### Рост канала
- Динамика подписчиков
- Отписки по периодам
- Прирост: откуда приходят новые подписчики (если доступно через API)

### Лучшие посты
- Топ по охватам
- Топ по вовлечённости (реакции + комментарии + репосты)
- Топ по реакциям

### По рубрикам
- Сравнение рубрик по среднему охвату
- Частота публикаций по рубрикам
- Динамика каждой рубрики во времени

### Временные паттерны
- Лучшее время суток для публикации
- Лучшие дни недели
- Тепловая карта активности аудитории

### Вовлечённость
- Реакции по типам
- Комментарии (количество, если канал открытый)
- Репосты
- ER (engagement rate) по постам

## Технические метрики ИИ

Находятся в профиле пользователя (технические настройки), не в основной навигации. Для тех, кто хочет контролировать использование и стоимость ИИ.

| Метрика | Описание |
|---|---|
| Стоимость токенов | Расходы на API по дням/неделям |
| Количество запросов | Нагрузка и паттерны использования |
| Оценки ответов | Статистика лайков/дизлайков на ответы ИИ |
| Активность агентов | Статус и история агентских задач (v2) |
| Использование по моделям | Распределение, если настроено несколько моделей |

---

## Связь аналитики с ИИ

Метрики канала доступны макро-ИИ как часть контекста. Это значит, что можно спросить в глобальном чате:

- "Почему охваты упали в последние две недели?"
- "Какая рубрика приносит больше всего новых подписчиков?"
- "В какое время мои посты получают больше всего просмотров?"

ИИ анализирует данные и даёт объяснения, а не просто показывает цифры.
