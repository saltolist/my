# Пространственная модель (актуально для demo)

Документ фиксирует, как пространственная логика реализована в текущем демо (`demo/`, Next.js + TS).

---

## Базовая идея

В демо используются два уровня работы:

- **Глобальное пространство (`global`)** — стартовый экран и сохранённые глобальные чаты.
- **Локальные пространства постов** — отдельное рабочее пространство каждого поста.

Главная цель модели — не смешивать задачи уровня канала и задачи уровня конкретного поста.

---

## Глобальное пространство (`global`)

Глобальный уровень в демо представлен двумя состояниями:

- `home` — новый пустой чат (ввод в центре экрана);
- `gchat` — открытый сохранённый глобальный чат.

**Что доступно на уровне интерфейса:**

- Отправка сообщений глобальному ИИ.
- Создание нового глобального чата через стартовый экран.
- Просмотр и удаление сохранённых глобальных чатов.
- Composer с выбором LLM-модели, Web Search-модели и режимом мультиответа.
- Универсальное attach-меню `+`: прикрепить пост (с фильтром уже привязанных), прикрепить файл с компьютера, медиа из прикреплённых к сообщению постов.

**Важно про пины в демо:** отдельной сущности «закреплённые посты чата» в UI нет. Привязка постов выполняется на уровне конкретного сообщения через composer, и медиа в attach-меню берётся именно из этих привязок, а не из глобального списка пинов.

---

## Локальное пространство поста

Открывается из ленты по клику на карточку поста.

Локальное пространство объединяет:

- чат поста с локальным ИИ;
- первое сообщение — карточка поста (TG-стиль медиа над текстом + текст), редактируется inline (`✏ Редактировать`): textarea авто-ресайзится, медиа остаётся сверху, есть кнопка `🖼 Добавить медиа`;
- статус поста (черновик / отложен / опубликован);
- метрики (для опубликованных);
- локальные заметки поста (режим `Заметки`).

Внутри пространства есть два режима: `chat` и `notes`, переключение через меню `•••`.

Composer в режиме `chat` имеет собственное attach-меню: `Прикрепить пост`, `Медиа из поста`, `Прикрепить файл с компьютера` и `Медиа из прикреплённых постов` (когда привязаны другие посты).

---

## Границы контекста в демо

- Локальный чат работает в рамках выбранного поста и его истории.
- Глобальный чат работает с контекстом уровня канала.
- Глобальные заметки имеют AI-флаг (`● ИИ` / `○ ИИ`).
- Локальные заметки доступны в пространстве поста и в общем разделе заметок (вкладка `Локальные`).

---

## Навигационная проекция пространств

- Логотип и кнопка `Новый чат` -> переход в `home`.
- `Лента` -> вход в локальные пространства постов.
- `Пространство поста` -> чат поста + заметки поста.
- `Чаты` -> список глобальных и локальных переписок.
