# Заметки

> **Актуальное поведение web-клиента** — в [10-pages.md](./10-pages.md) (§5, §7) и [wireframes/04-note.md](../wireframes/04-note.md). Ниже — продуктовая модель с пометками, где текущая local-first фаза расходится с целевым видением.

Заметки — намеренно открытый инструмент. Платформа не диктует, как их использовать. Это свободное пространство, которое автор наполняет так, как считает нужным.

---

## В web-клиенте (local-first)

- Каталог `notes`: scope `Все` / `Глобальные` / `Локальные` (стартовый — `Все`); фильтр по AI-флагу; кнопка `+ Новая заметка` в scope `Все` и `Глобальные`.
- Локальные заметки **также** в общем каталоге (scope `Локальные` / `Все`) и в раскрывающемся списке под `Заметки` в сайдбаре.
- AI-флаг (`● ИИ` / `○ ИИ`) есть **и у глобальных, и у локальных** заметок (карточка, режим `notes` поста, меню `•••` на странице заметки).
- Крошки локальной заметки: `Лента / Название поста / Название`.
- Drag-and-drop: файл из панели вложений → тело; перестановка embed-чипов в режиме `edit`.

---

## Два уровня заметок (целевая модель)

### Локальные заметки (локальный уровень)

Привязаны к конкретному посту. Создаются и просматриваются в пространстве поста (режим `notes` в шапке).

- В целевой модели видны только внутри пространства того поста
- В web-клиенте дополнительно доступны в общем каталоге «Заметки»
- Попадают в контекст локального ИИ поста (в web-клиенте — с учётом AI-флага на карточке)
- Подходят для: черновиков фрагментов, идей для продолжения, фактов под конкретный пост

### Глобальные заметки (глобальный уровень)

Не привязаны к конкретному посту. Хранятся в общем разделе "Заметки" в навигации.

- Доступны из любого места платформы
- Могут быть добавлены в контекст глобального ИИ через флаг "включить в контекст"
- Подходят для: идейника, базы знаний, стратегии канала, исследований, личных заметок

---

## Флаг "включить в контекст"

**В web-клиенте** переключатель есть у глобальных и локальных заметок.

**В целевой модели** — только у глобальных. Управляет тем, что глобальный ИИ знает о канале:

- Без флага — заметка видна только пользователю, глобальный ИИ её не учитывает
- С флагом — содержание передаётся в контекст глобального ИИ

Локальные заметки в целевой модели всегда участвуют в контексте локального ИИ своего поста; в web-клиенте флаг на локальной заметке дополнительно фильтрует каталог и влияет на отображение `● ИИ` / `○ ИИ`.

## Возможные сценарии использования

Ниже — примеры. Это не ограничение, а иллюстрация.

### Черновики и фрагменты (локальные)
Набросок идеи под конкретный пост, незавершённый абзац, альтернативный вариант текста.

### База знаний для ИИ (глобальные, с флагом)
Факты, данные, исследования, личные наблюдения — то, что ИИ должен учитывать при работе с любым постом.

### Идейник (глобальные)
Темы для будущих постов, вопросы от читателей, интересные наблюдения — сырой материал для развития.

### Структура канала (глобальные, с флагом)
Описание рубрик в свободной форме, план развития канала, связи между темами.

### Мониторинг (глобальные)
Ссылки на интересные посты других каналов, референсы, разборы чужих текстов.

### Личные заметки (глобальные, без флага)
Мысли, дневник автора, рефлексия — то, что не связано с каналом напрямую.

## Формат

Заметки поддерживают базовое форматирование (Markdown). Возможность вставлять ссылки, изображения и делать структурированные списки — на усмотрение реализации.
