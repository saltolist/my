# entities/note

Note entity: unified card UI for global and local notes, AI context toggle.

## Public API

- `NoteCard` — list card (`variant: "global" | "local"`)
- `NoteAiToggle` — badge or switch toggle for AI context
- Re-export hooks from `model/useGlobalNotes.ts` via entity consumers
