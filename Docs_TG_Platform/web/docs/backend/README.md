## Текущее состояние

Фронтенд **полностью готов к подключению бэкенда**:
- Repository Pattern изолирует все запросы
- Zod-схемы описывают ожидаемые форматы данных
- MSW имитирует все эндпоинты
- Типы ID — string (UUID-совместимые)
- Все даты — ISO-8601

Для запуска без бэкенда используется MSW. Переключение на реальный API — одна переменная окружения.
