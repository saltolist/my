# Профиль (`profile`)

Экран разделён на три вкладки в шапке: `Настройки`, `Канал`, `Аналитика платформы` (стартовая — `Канал`).

```
┌─────────────────────────────────────────────────────────────────┐
│  Профиль канала    [Настр.|Канал|Аналит.]           [← Назад]   │
│ ─────────────────────────────────────────────────────────────── │
│  (содержимое активной вкладки)                                  │
└─────────────────────────────────────────────────────────────────┘
```

## Шапка

- Заголовок `Профиль канала` (`PageHeader`).
- Вкладки раздела — в центральном слоте шапки (рядом с поиском на других экранах здесь только вкладки).
- Справа: `← Назад` → `home`.

При уходе с вкладки `Настройки` с несохранёнными изменениями — `window.confirm`.

## Вкладка `Канал`

Редактируемая модель профиля канала:

- Целевая аудитория: демография, психография, поведение, сегменты;
- Рубрики: название, описание, частота, стиль, примеры, теги;
- Тон и голос;
- Структурные паттерны;
- Запрещённые темы, слова и форматы;
- Бренд-бук;
- Конкуренты и референсы.

Полная модель — в [`05-channel-profile.md`](../../product/05-channel-profile.md).

## Вкладка `Настройки`

### Тема оформления

Сегментированный переключатель из трёх кнопок: `☀ Светлая`, `🖥 Системная`, `🌙 Тёмная`. Выбор сохраняется в `localStorage` и применяется через атрибут `data-theme` на `<html>`; режим `Системная` следует `prefers-color-scheme`.

### ИИ-движок

- Несколько LLM-моделей: для каждой — выбор провайдера и модели через `ModelPicker` (пилюля с иконкой `🧠` и dropdown’ом), поле API-ключа с явной рамкой, флаг `Включить в мультиответ`, кнопка «сделать активной».
- Несколько Web Search моделей — аналогичный блок с иконкой `🔎`.
- Общий переключатель `Мультиответ` (включается, если у нескольких моделей включён флаг `Включить в мультиответ`).
- Системный промпт — textarea с явной рамкой.

Параметры `температура` и `max tokens` в web-клиенте пока не вынесены в UI.

### Telegram

MTProto-форма:

- `api_id`, `api_hash`;
- телефон;
- код подтверждения;
- `sessionName`;
- выбор `syncMode` через `ModelPicker`;
- кнопки подключения/отключения канала;
- `Сбросить настройки` — возврат мок-состояния Telegram-формы.

Все поля оформлены с явными рамками и фоном.

## Вкладка `Аналитика платформы`

Секции:

- Использование ИИ: запросы, токены, стоимость;
- Активность: чаты, заметки, посты.

## Примечание

В web-клиенте вкладка `Аналитика платформы` пока не включает историю агентских действий и отдельный блок оценок лайк/дизлайк.
