# Глобальный чат (`gchat`)

**Reference:** `web-legacy/src/screens/gchat/`, `web-legacy/src/widgets/chat-thread/`

## Состояние экрана

```
┌─────────────────────────────────────────────────────────────────┐
│ [☰] Чаты / Название чата                    [← Назад] [•••]     │
├─────────────────────────────────────────────────────────────────┤
│ [Я] ...                                                         │
│ [ИИ] ...                                                        │
│ ┌─────────────────────────────────────────────────────────────┐ │
│ │ Написать сообщение...                                       │ │
│ │ [+] [🧠 GPT-4.1 ▾] [🔎 Web ▾]                          [↑]  │ │
│ └─────────────────────────────────────────────────────────────┘ │
└─────────────────────────────────────────────────────────────────┘
```

## Шапка

- Слева: **`PageHeaderMenuButton`** (hamburger) + хлебные крошки `Чаты / Название чата`.
- Справа: `← Назад` (→ каталог чатов), меню `•••` → **`Удалить чат`**.
- Для omnichannel-чатов пункт удаления **скрыт**.

## Сообщения

- Ответы ИИ: одиночный или мультиответ; `AiMessageToolbar` (копирование, подсказка по моделям).
- Пользовательские: редактирование по hover, ветки версий (`userBranches`), стрелки переключения.
- После правки — stub-ответ с пометкой «отредактировано».

## Composer

| Элемент | Действие |
|---|---|
| Textarea | Авто-ресайз до 16 строк |
| `+` | Attach (**вверх**): `Прикрепить файл`, `Медиа из прикреплённых постов` (если привязаны посты с медиа). **Нет «Прикрепить пост»** — пост через `@` |
| `🧠 …` / `🔎 …` | ModelPicker + pill `Мультиответ` |
| `↑` | Отправка сообщения |

## Прикрепления

- Чип поста/файла над textarea; удаление крестиком или Backspace.
- `@<title>;` → чип-пост.
- Отдельного блока закреплённых постов над composer **нет**.
