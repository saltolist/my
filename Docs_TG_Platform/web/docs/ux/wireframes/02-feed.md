## Шапка

- Заголовок `Лента` слева.
- В центре: поиск по постам и переключатель ширины колонки.
- Справа: `← Назад` → `home`.

## Ширина колонки

Кнопки показывают **Компьютер / Планшет / Телефон** (не числа px). Значения: 500 / 390 / 270 px (`feedPostWidth.ts`).

## Структура ленты

Три фиксированные секции сверху вниз:

| Секция | Порядок внутри |
|---|---|
| **Опубликованные** | Группы по дням (`feed-day-marker`), внутри группы — по дате публикации |
| **Отложенные** | По дате запланированной публикации, ближайшие сверху |
| **Черновики** | Порядок задаётся drag-and-drop |

Фильтров по статусу нет — все три секции всегда видны.

## Карточка поста

| Элемент | Описание |
|---|---|
| Медиа | TG-стиль блок над текстом (`PostMediaBlock`) |
| Текст | Полный текст без обрезки; пустой черновик — «Пост пустой — нажми чтобы начать писать» |
| Статус | SVG-иконка + текст: ✓ Опубликован / ◷ Отложено / ✎ Черновик |
| Метрики (опубликованные) | Реакции-пилюли + просмотры/репосты в footer |
| Комментарии | Отдельная строка `PostCommentsRow` под footer (не в одной линии с метриками) |
| Рубрика | **Не отображается** на карточке ленты |
| Клик | Открывает пространство поста |

## Drag-and-drop черновиков

- Слева — вертикальная рель с **6-точечным** drag-handle (не ⋮⋮).
- Тянуть только за handle.
- Авто-скролл у краёв экрана и блока ленты.
- Порядок сохраняется в in-memory domain store (до перезагрузки вкладки).

## Composer внизу ленты

- Placeholder `Написать пост...`.
- Авто-ресайз до **16 строк**.
- `+` сразу открывает file picker — медиа к черновику, TG-стиль над textarea.
- LLM/Web Search **не показываются** (`scope="feed"`).
- `↑` создаёт черновик в секции `Черновики`.

## Название поста в хлебных крошках

Первое предложение первой строки до `.`; если точки нет — вся первая строка до `\n`. Подробнее — в `03-post.md`.
