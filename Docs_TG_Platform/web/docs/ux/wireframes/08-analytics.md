## Шапка

- Заголовок `Аналитика канала`.
- Справа: `← Назад` → `home`.
- **Период не в шапке** (на desktop) — picker внутри карточки «Динамика прироста».
- На mobile период — `PageHeaderSelect` в шапке.

## Периоды

**5 вариантов:** `24 часа`, `7 дней`, `30 дней`, `90 дней`, `Всё время` (`web/src/shared/data/analytics-seed.ts` → `ANALYTICS_PERIOD_OPTIONS`).

> **M2 scaffold:** период в шапке на mobile (`PageHeaderSelect`); на desktop — `AnalyticsPeriodFilter` в шапке до появления M3 dashboard с picker внутри карточки «Динамика прироста».

## Блоки экрана

| Блок | Описание |
|---|---|
| **Динамика прироста** | Summary mini-metrics + `MultiSeriesTrendChart`; selector видимых серий метрик |
| **Прирост по метрикам** | `ChannelMetricBars` |
| **Реакции** | `ChannelReactionsPanel` |
| **Тепловая карта активности** | `AnalyticsHeatmap` |
| **Лучшие посты за период** | Таблица с переходом к посту по `→` |

## Метрики в графике

Просмотры, подписчики, реакции, комментарии, репосты, ER — через selector серий (`channelAnalyticsTrend.ts`).

Данные — mock/seed (`analytics-seed.ts`).
