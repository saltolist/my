# Home (новый глобальный чат)

> Отдельной страницы «Главная» нет. Стартовый экран — `home` (`/`). Открывается кликом по логотипу или пункту `Глобальный чат` в сайдбаре.

**Reference:** `web-legacy/src/screens/home/ui/HomePage.tsx`

## PageHeader

- Слева — кнопка-бренд `TG Platform` (повторный переход на `home`).
- Отдельного заголовка экрана нет.

## Sidebar

| Элемент | Поведение |
|---|---|
| Логотип `✦ TG Platform` | Переход в `home` |
| `Глобальный чат` | Иконка `+`, активен на `screen === home` |
| Порядок пунктов | **Глобальный чат → Аналитика → Лента** → (строка поста, если открыт) → Заметки ▾ → Чаты ▾ → Профиль |
| Сворачивание панели | Отдельная кнопка rail в `SidebarHeader`, не клик по логотипу |

## Composer (`scope="home"`)

| Элемент | Действие |
|---|---|
| Поле ввода | Placeholder: `Сообщение... введите @ чтобы прикрепить пост`; авто-ресайз вверх до **10 строк**, дальше скролл |
| `↑` | Создаёт новый глобальный чат и открывает `gchat` |
| `+` | Attach-меню (**раскрывается вниз**): `Прикрепить файл`, `Медиа из прикреплённых постов` (если к сообщению уже привязан пост с медиа) |
| `🧠 …` | `ModelPicker` LLM-модели |
| `🔎 …` | `ModelPicker` Web Search; рядом pill `Мультиответ`, если в профиле включено у нескольких моделей |

## Прикрепление постов

- **Не через `+` меню.** Пост прикрепляется через `@` — выпадающий список постов ленты (`ComposerMentionDropdown`).
- Синтаксис `@<title>;` в тексте превращается в чип-пост.
- Выбранный пост или файл — чип над textarea; удаление крестиком или Backspace при пустом поле.
- Медиа в подменю — миниатюры 72×72, адаптивная сетка (1–3 в ряд, 4 — 2×2, 5+ — три колонки со скроллом).
