## Шапка

- Заголовок `Чаты` слева.
- В центре: поиск + **dropdown** scope `Все / Глобальные / Локальные` (`PageHeaderSelect` на desktop; на mobile — в шапке отдельно).
- Стартовый scope: `Все`.
- Справа: `← Назад` → `home`.

## Кнопка «Новый чат»

- Показывается на вкладках **`Все`** и **`Глобальные`** в `FilterToolbar` над списком.
- Переход на `home` для создания нового глобального чата.

## Карточки

| Тип | Содержание |
|---|---|
| Глобальный | Название (первый запрос), превью user/assistant строк, дата; icon rail: 💬 или ✈ (omnichannel) |
| Локальный | То же + icon rail 📄 (feed); tooltip `Пост: {название}` |

## Действия

| Действие | Поведение |
|---|---|
| Клик по карточке | Глобальный → `gchat`; локальный → `post/{id}/?chat={chatId}` |
| `→ пост` | **Нет отдельной кнопки** — переход к посту через клик по локальной карточке |
| Удаление | Context menu (`ChatListCardMenu`), не видимая `🗑` на карточке |

## Вкладка «Все»

Глобальные сверху, затем локальные.

## Вкладка «Локальные»

Каждая карточка — **один локальный чат** (не пост). У одного поста может быть несколько карточек.
