**Reference:** `web-legacy/src/screens/post/`, `web-legacy/src/widgets/post-workspace/`

├─────────────────────────────────────────────────────────────────┤

## Шапка

- Слева — хлебные крошки `Лента / …` (до трёх уровней).
- Справа: `↑ К посту`, кластеры `Заметки` и `Чаты`, `← Назад`, меню `•••`.
- `Заметки` / `Чаты` — переключатели режимов; повторный клик возвращает в `chat`.
- В режимах `chats`, `notes`, `comments` — поиск по центру шапки.
- **`+ Новая заметка` / `+ Новый чат` — не в шапке**, а в `FilterToolbar` под шапкой (см. состояния C/D).
- Шапка фиксирована (`page-header--post`), контент скроллится под неё.

## Навигация «← Назад»

Использует **`router.back()`** / parent path, не внутренний `postViewStack`. Стек `postViewStack` в store не применяется для навигации назад.

## Карточка поста (режим просмотра)

- Метрики: реакции, просмотры, репосты.
- Действия — **иконки** «Скопировать» и «Редактировать» (`PostCardToolbar`), не текстовая кнопка `✏ Редактировать`.
- Строка «Комментарии (N) ›» — для опубликованных.

## Локальный чат

- При входе `currentPostChatId === null`, composer пуст, видна только карточка поста.
- Первое сообщение создаёт локальный чат (заголовок ~40 символов первого сообщения).
- Память между разными локальными чатами одного поста не делится.

## Состояние B — Inline-редактирование

- TG-стиль медиа-блок сверху.
- Кнопка прикрепления медиа — **иконка «Прикрепить файл»** (не «🖼 Добавить медиа»).
- Textarea без min-height, растёт по контенту.
- Кнопки внизу: **`Сохранить`**, затем **`Отмена`**.

## Состояние C — Режим заметок

```
┌─────────────────────────────────────────────────────────────────┐
│ Лента / Название / Заметки  [🔎 Поиск]  [Заметки*][Чаты] [←][•••]│
├─────────────────────────────────────────────────────────────────┤
│ [Все|В контексте ИИ|Не в контексте]           [+ Новая заметка] │
│ ┌─────────────────────────────────────────────────────────┐     │
│ │ Заголовок                                    [🧠 меню]  │     │
│ │ Превью...     [🧠 В контексте / Не в контексте]         │     │
│ └─────────────────────────────────────────────────────────┘     │
└─────────────────────────────────────────────────────────────────┘
```

- Сетка локальных заметок; клик → страница заметки.
- AI-флаг: кнопка с иконкой мозга + текст **«В контексте» / «Не в контексте»** (не `●/○ ИИ`).
- Удаление — через context menu на карточке.
- Фильтр **Все / В контексте ИИ / Не в контексте** в `FilterToolbar`.

## Состояние D — Список локальных чатов

```
┌─────────────────────────────────────────────────────────────────┐
│ Лента / Название / Чаты     [🔎 Поиск]  [Заметки][Чаты*] [←]   │
├─────────────────────────────────────────────────────────────────┤
│ [Все|В контексте ИИ|Не в контексте]              [+ Новый чат]  │
│ ┌─────────────────────────────────────────────────────────┐     │
│ │ Перепиши пост            30 апр 2026          [меню 🗑] │     │
│ │ "Сделай его более..."                                   │     │
│ └─────────────────────────────────────────────────────────┘     │
└─────────────────────────────────────────────────────────────────┘
```

- `+ Новый чат` в `FilterToolbar` — пустой черновик, переключение в `chat`.
- Клик по карточке → соответствующий чат в режиме `chat`.
- Удаление — context menu.

## Состояние E — Комментарии (только опубликованные)

Вход: строка «Комментарии» на карточке, `PostCommentsRow` в ленте или из режима `chat`.

- Поиск фильтрует список.
- `Ответить` — цитата автора в composer.
- Новые комментарии — in-memory.

## Меню `•••`

Общие пункты для всех статусов:

- `Новый чат`
- `Новая заметка`

### Черновик

- `Опубликовать`
- `Запланировать`
- `Удалить`

### Отложенный

- `Опубликовать`
- **`Перенести публикацию`**
- `Отменить публикацию`
- `Удалить`

### Опубликованный

- `Удалить`

## Composer (режим `chat`)

| Элемент | Действие |
|---|---|
| Textarea | Авто-ресайз до 16 строк |
| `+` | Attach (**вверх**): `Медиа из поста` (если есть), `Медиа из прикреплённых постов`, **`Загрузить с компьютера`**. Пункта «Прикрепить пост» **нет** — пост через `@` |
| `🧠 …` / `🔎 …` | ModelPicker + pill `Мультиответ` |
| `↑` | Отправить (или создать чат, если активного нет) |
