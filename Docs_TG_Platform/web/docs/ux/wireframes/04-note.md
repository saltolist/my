# Страница заметки (`note`)

**Reference:** `web-legacy/src/screens/note/`, `web-legacy/src/widgets/note-editor/`

Заметка поддерживает два режима: `view` и `edit`.

## Шапка

```
┌─────────────────────────────────────────────────────────────────┐
│ Заметки / Название          или   Лента / Пост / Название       │
│                                              [← Назад] [•••]      │
└─────────────────────────────────────────────────────────────────┘
```

- Слева — хлебные крошки:
  - глобальная: `Заметки / Название`;
  - локальная: `Лента / Название поста / Название` (средняя часть — ссылка на пост).
- Справа: `← Назад`, overflow-меню `•••`.
- Меню: `Учитывать в ИИ` / `Не учитывать в ИИ`, `Удалить заметку` / `Отменить` (для новой).

## Заголовок

- Textarea с авто-подгонкой ширины шрифта под одну строку (`useFitTitleSize`).
- **В режиме `view` textarea остаётся редактируемой** — режим влияет на тулбар и тело, не на readOnly заголовка.

## Режим `view`

- Тулбар: иконка «редактировать», `Сохранить` (неактивна без изменений).
- Отрендеренный текст (embed-чипы файлов/медиа).
- Блок `Вложения` (если есть).
- Дата создания/изменения внизу.
- AI-флаг — только в меню `•••`, не в теле.
- **Нет** метки `📌 Локальная` и ссылки `→ пост` в теле страницы (есть только в breadcrumbs для локальной заметки).

## Режим `edit`

- Тулбар: `📎` (файл), иконка «просмотр», `Сохранить` (активна при изменениях).
- **`Отменить`** — только когда есть несохранённые изменения (`NoteHeaderToolbar`).
- Редактируемое тело (`NoteBodyEditor`).
- Блок `Вложения` с drag-and-drop.

## Drag-and-drop в заметке

- Файл из `Вложения` → embed-чип в теле.
- Embed-чипы можно переставлять (только в `edit`).
- Drop файлов с рабочего стола в тело добавляет вложения.

## Поведение переходов

- `beforeunload` + confirm при уходе с несохранёнными изменениями.
- `Сохранить` активна только при `changed`.
