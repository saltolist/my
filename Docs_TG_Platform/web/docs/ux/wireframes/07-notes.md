# Заметки (`notes`)

**Reference:** `web-legacy/src/screens/notes/`

Экран объединяет глобальные и локальные заметки.

```
┌─────────────────────────────────────────────────────────────────┐
│  Заметки   [🔎 Поиск] [Все ▾]                       [← Назад]   │
│ ─────────────────────────────────────────────────────────────── │
│  [Все] [В контексте ИИ] [Не в контексте]      [+ Новая заметка] │
│ ─────────────────────────────────────────────────────────────── │
│  ┌────────────────────────┐  ┌────────────────────────┐         │
│  │ Заголовок         [⋮]  │  │ Заголовок         [⋮]  │         │
│  │ Превью тела...         │  │ Превью тела...         │         │
│  │ [🧠 В контексте]       │  │ [🧠 Не в контексте]    │         │
│  │           1 мая · Глоб.│  │     30 апр · Локальная │         │
│  └────────────────────────┘  └────────────────────────┘         │
└─────────────────────────────────────────────────────────────────┘
```

## Шапка

- Заголовок `Заметки` слева.
- В центре: поиск + **dropdown** scope `Все / Глобальные / Локальные` (`PageHeaderSelect` на desktop).
- Стартовый scope: `Все`.
- Справа: `← Назад` → `home`.

## Строка фильтров

- Фильтр `Все / В контексте ИИ / Не в контексте`.
- `+ Новая заметка` — в scope `Все` и `Глобальные`.

## Карточка заметки

| Элемент | Описание |
|---|---|
| Заголовок | Название заметки |
| Превью | Фрагмент тела |
| AI-флаг | Кнопка: иконка 🧠 + **«В контексте» / «Не в контексте»** (не `●/○ ИИ`) |
| Мета | `{дата} · Глобальная` или `{дата} · Локальная` |
| Удаление | Context menu (`NoteListCardMenu`) |
| `→ пост` | **Нет** на карточке списка |

## Действия

- Клик по карточке → страница заметки.
- Фильтр и поиск — в реальном времени.
