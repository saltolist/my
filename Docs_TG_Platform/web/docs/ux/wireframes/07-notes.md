# Заметки (`notes`)

Экран объединяет глобальные и локальные заметки.

## Структура

```
┌─────────────────────────────────────────────────────────────────┐
│  Заметки   [🔎 Поиск] [Все|Глоб.|Лок.]              [← Назад]   │
│ ─────────────────────────────────────────────────────────────── │
│  [Все] [В контексте ИИ] [Не в контексте]      [+ Новая заметка] │
│ ─────────────────────────────────────────────────────────────── │
│  ┌────────────────────────┐  ┌────────────────────────┐         │
│  │ Заголовок              │  │ Заголовок              │         │
│  │ Превью тела...         │  │ Превью тела...         │         │
│  │ ● ИИ           [🗑]    │  │ ● ИИ  📌 Лок. → пост    │         │
│  └────────────────────────┘  └────────────────────────┘         │
└─────────────────────────────────────────────────────────────────┘
```

## Шапка

- Заголовок `Заметки` слева.
- В центре: поиск и вкладки scope `Все` / `Глобальные` / `Локальные` (стартовая — `Все`).
- Справа: `← Назад` → `home`.

## Строка фильтров

- Фильтр `Все / В контексте ИИ / Не в контексте`.
- Кнопка `+ Новая заметка` в правом крае (видна в scope `Все` и `Глобальные`).

## Карточка заметки

| Элемент | Описание |
|---|---|
| Заголовок | Название заметки |
| Превью | Фрагмент тела |
| `● ИИ / ○ ИИ` | Переключатель AI-флага (у глобальных и локальных) |
| `🗑` | Удаление с подтверждением |
| `📌 Локальная · → пост` | Только для локальной заметки |

## Действия

- Клик по карточке → открывает страницу заметки.
- Клик по `→ пост` → открывает пространство поста.
- Фильтр и поиск работают в реальном времени.
