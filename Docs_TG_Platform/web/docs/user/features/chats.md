# Чаты

Чаты — AI-диалоги в платформе. Бывают двух видов: **глобальные** и **привязанные к посту**.

## Глобальные чаты

Доступны из раздела **«Чаты»** в sidebar.

**Используются для:**
- Мозгового штурма
- Разработки стратегии канала
- Общих вопросов к AI без привязки к конкретному посту

**Как создать:**
1. Нажмите **«+»** в разделе Чаты.
2. Появится новый диалог.
3. Введите сообщение — AI ответит с учётом профиля вашего канала.

### Омниканальный чат
Особый тип глобального чата, который объединяет контекст всех активных каналов (если подключено несколько).

## Чаты к постам

Привязаны к конкретному посту и доступны из рабочей области поста (вкладка **«Чаты»**).

**Используются для:**
- Обсуждения конкретного поста с AI
- Генерации вариантов текста в контексте поста
- Работы с конкретным разделом или идеей

**Как создать:**
1. Откройте пост из ленты.
2. Перейдите на вкладку **«Чаты»**.
3. Нажмите **«+»** — создастся новый чат.
4. Или начните диалог сразу из редактора нажав кнопку AI.

## Интерфейс чата

```
┌──────────────────────────────────────────┐
│  История сообщений                       │
│                                          │
│  👤 Вы: ...текст запроса...              │
│  🤖 AI: ...текст ответа...               │
│      [LLM: OpenAI / gpt-4o]              │
│      [Web: Perplexity / search-api]      │
│                                          │
├──────────────────────────────────────────┤
│  ✏️ Введите сообщение...        [AI ↑]   │
└──────────────────────────────────────────┘
```

## Варианты ответов (Multi-mode)

Если в настройках AI включён **Multi-response**, AI может предложить несколько вариантов ответа одновременно. Вы выбираете лучший.

## Ветки диалога

Если вы хотите попробовать другой вариант вопроса без потери истории:
- Нажмите на стрелку рядом с вашим сообщением — откроется ветка.
- Введите альтернативный текст.
- Переключайтесь между ветками.

## Управление чатами

- **Переименовать** — правый клик на чате в sidebar или меню чата
- **Удалить** — через меню чата (необратимо)
