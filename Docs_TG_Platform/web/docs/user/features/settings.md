# Настройки

Раздел настроек содержит три блока: **Профиль канала**, **AI-модели** и **Telegram**.

---

## Профиль канала

Здесь описывается ваш канал — эта информация используется AI при генерации контента.

### Блок «Суть канала»
| Поле | Описание |
|------|----------|
| Тема | О чём ваш канал |
| Аудитория | Кто ваши читатели |
| Обещание | Что получает подписчик |
| Угол | Уникальный взгляд / подача |
| Автор | Кто пишет (имя/роль) |

### Блок «Голос и стиль»
- **Тон** — как вы общаетесь (официально, неформально, экспертно)
- **Формат** — типичная структура постов
- **Фразы** — характерные выражения, речевые обороты

### Блок «Правила»
- **Обязательно** — что всегда должно быть в постах
- **Запрещено** — чего никогда не должно быть

### Рубрики
Рубрики — категории постов. AI учитывает рубрику при генерации.
- Добавьте рубрику через кнопку **«+»**
- Задайте название и описание
- Рубрика будет доступна при создании поста

## AI-модели

Настройка моделей для различных задач:

| Тип модели | Назначение |
|-----------|-----------|
| LLM | Основной генератор текста |
| Веб-поиск | Поиск актуальной информации |
| Vision | Анализ изображений |
| Генерация изображений | Создание иллюстраций |
| Оркестратор | Управление цепочкой агентов |
| Веб-резонёр | Анализ с поиском в вебе |
| RAG-резонёр | Анализ документов |

Для каждой модели указывается: провайдер, название модели, API-ключ.

Флаг **«Multi-response»** включает генерацию нескольких вариантов ответа одновременно.

**Системный промпт** — текст, который отправляется перед каждым запросом к AI.

## Telegram

Подключение к Telegram для синхронизации и публикации.

### Авторизация Telegram-аккаунта
1. Введите **API ID** и **API Hash** (с [my.telegram.org](https://my.telegram.org)).
2. Введите номер телефона.
3. Введите код из Telegram.

### Подключение канала
1. Введите **@handle** канала (или ссылку).
2. Выберите **режим синхронизации**:
   - `live-only` — только новые посты
   - `history-and-live` — история и новые посты
   - `publish-only` — только публикация через платформу

### Telegram Bot
Для публикации и интерактивных функций можно подключить бота:
1. Создайте бота через [@BotFather](https://t.me/BotFather).
2. Скопируйте API-токен и вставьте в поле.
3. Нажмите «Подключить».
