# Заметки

Заметки — текстовые записи, которые можно вести как отдельно (глобально), так и в контексте конкретного поста.

## Виды заметок

| Тип | Где хранится | Для чего |
|-----|-------------|----------|
| Глобальная | Раздел «Заметки» | Идеи, шаблоны, справочная информация |
| К посту | Внутри поста, вкладка «Заметки» | Наброски, источники, заметки по конкретному посту |

## Глобальные заметки

**Как создать:**
1. Перейдите в раздел **«Заметки»** в sidebar.
2. Нажмите **«+»**.
3. Введите заголовок и текст.

**Сетка или список:**  
На широких экранах заметки отображаются сеткой, на мобильных — списком.

## Заметки к посту

**Как создать:**
1. Откройте пост.
2. Перейдите на вкладку **«Заметки»**.
3. Нажмите **«+»** — откроется редактор заметки.

## Редактор заметок

- Поддерживает свободный текстовый ввод.
- Можно прикрепить **файлы** (изображения, документы).
- Переключатель **AI** — включает AI-режим заметки (AI может её дополнять/анализировать).

## AI-режим заметки

Если у заметки включён флаг **AI**, она участвует в AI-контексте чатов к посту:
- AI учитывает содержимое заметки при генерации ответов.
- Удобно для заметок с источниками, фактами или требованиями к посту.

## Управление заметками

- **Переименовать** — через меню в списке
- **Удалить** — через меню (необратимо)
- Заметки автосохраняются при вводе
