# Аналитика

Раздел **Аналитика** показывает статистику вашего Telegram-канала.

## Что показывается

### Топ-посты
Таблица самых просматриваемых постов за период с колонками:
- Превью текста поста
- Просмотры
- Репосты
- Реакции

### Метрики канала
- Общее число просмотров за период
- Прирост подписчиков
- Активность аудитории по дням

## Как использовать

1. Перейдите в раздел **«Аналитика»** через sidebar.
2. Выберите временной период (день / неделя / месяц).
3. Изучите топ-посты — найдите форматы, которые работают лучше всего.
4. Нажмите на пост в таблице — перейдёт в рабочую область поста.

## Подсказки по интерпретации

- **Высокие просмотры + мало реакций** — контент читают, но не вовлекает.
- **Высокие реакции + мало просмотров** — узкая аудитория, но лояльная.
- **Много репостов** — виральный контент, стоит повторить формат.

## Требования

Аналитика работает только при **подключённом Telegram-канале** в режиме синхронизации «история + живые данные».
