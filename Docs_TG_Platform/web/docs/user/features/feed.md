# Лента и посты

Лента — центральный раздел платформы. Здесь собраны все посты канала: черновики, запланированные и опубликованные.

## Структура ленты

```
Лента
├── Черновики          — посты в работе, можно сортировать перетаскиванием
├── Запланированные    — посты с конкретной датой публикации
└── Опубликованные     — архив опубликованных постов
```

## Карточка поста

Каждая карточка показывает:
- **Статус** (черновик / запланирован / опубликован)
- **Рубрику** — если задана в настройках канала
- **Превью текста**
- **Дату** — когда запланирован или опубликован
- **Метрики** — просмотры и реакции (для опубликованных)

## Создание нового поста

1. Нажмите **«+»** в разделе «Черновики».
2. Откроется [редактор (Composer)](composer.md).
3. Напишите текст, добавьте медиа при необходимости.
4. Сохраните как черновик или сразу запланируйте.

## Статусы поста

| Статус | Описание |
|--------|----------|
| `draft` | Черновик — видит только автор |
| `scheduled` | Запланирован — дата публикации задана |
| `published` | Опубликован — виден в канале |

## Сортировка черновиков

Черновики можно переставлять **перетаскиванием** — порядок определяет приоритет работы над постами. Позиции сохраняются при перезагрузке.

## Контекстное меню поста

Правый клик / длинный тап на карточке открывает меню:
- **Открыть** — перейти в рабочую область поста
- **Дублировать** — создать копию поста
- **Запланировать** — задать дату и время
- **Опубликовать** — сменить статус на published
- **Удалить** — безвозвратно удалить пост

## Фильтрация

Лента поддерживает фильтрацию по рубрикам и статусу через панель фильтров вверху.
