# Редактор (Composer)

Composer — редактор для создания и редактирования постов. Открывается при создании нового поста или при переходе в рабочую область существующего.

## Основные элементы

```
┌──────────────────────────────────────────────┐
│  Текстовое поле                              │
│  (Markdown-подобный ввод)                    │
│                                              │
├──────────────────────────────────────────────┤
│  📎 Вложения                                 │
│  [Медиа] [Пост] [Файл]                       │
├──────────────────────────────────────────────┤
│  [AI] [Отправить]                            │
└──────────────────────────────────────────────┘
```

## Написание текста

- Поддерживается форматирование в стиле Telegram: `**жирный**`, `_курсив_`, `[ссылка](url)`.
- Для упоминания поста из ленты используйте `@` — откроется выпадающий список доступных постов.
- Текст сохраняется в черновик автоматически при каждом изменении.

## Вложения

### Медиа из поста
Прикрепите изображение или видео из другого поста:
1. Нажмите иконку 📎 → **«Медиа из поста»**.
2. Выберите пост — откроется галерея его медиафайлов.
3. Выберите нужный файл — он появится как вложение.

### Прикрепление поста
Позволяет процитировать или сослаться на пост:
1. Нажмите иконку 📎 → **«Пост»**.
2. Введите часть заголовка — поиск по закреплённым постам.
3. Выберите пост.

### Файлы
Загрузите произвольный файл с устройства.

## AI-генерация

1. Введите запрос или выделите фрагмент текста.
2. Нажмите кнопку **AI** (или иконку в тулбаре).
3. Редактор отправит запрос AI-ассистенту.
4. AI предложит один или несколько вариантов текста.
5. Выберите вариант или попросите переделать.

Контекст AI-запроса определяется областью Composer:
- **Home** — общий диалог, нет привязки к посту
- **Пост** — AI учитывает профиль канала и рубрику поста

## Публикация

После написания текста доступны действия:
- **Отправить как черновик** — сохранить и вернуться в ленту
- **Запланировать** — выбрать дату и время публикации
- **Опубликовать сейчас** — немедленно опубликовать
