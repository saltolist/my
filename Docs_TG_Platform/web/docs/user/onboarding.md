# Начало работы

## 1. Регистрация и вход

1. Откройте платформу в браузере.
2. На экране входа нажмите **«Зарегистрироваться»**.
3. Введите email и пароль — аккаунт создаётся мгновенно.
4. После входа вы попадаете на **главный экран (Home)** с AI-чатом.

> В демо-режиме можно войти без регистрации — данные сохраняются локально в рамках сессии.

## 2. Обзор интерфейса

```
┌─────────────────────────────────────────────────────┐
│  Sidebar (левая панель)   │   Основная область       │
│  • Home (AI-чат)          │                          │
│  • Лента                  │   Контент текущего       │
│  • Чаты                   │   экрана                 │
│  • Заметки                │                          │
│  • Аналитика              │                          │
│  • Настройки              │                          │
└─────────────────────────────────────────────────────┘
```

**Sidebar** — навигация между разделами. Переключается стрелкой или кнопкой «меню» на мобильном.

## 3. Подключение Telegram-канала

Чтобы видеть реальные данные и публиковать посты:

1. Перейдите в **Настройки → Telegram**.
2. Введите **API ID** и **API Hash** (получить на [my.telegram.org](https://my.telegram.org)).
3. Введите номер телефона — придёт код подтверждения.
4. После авторизации укажите **@handle канала** и нажмите «Подключить».
5. Выберите режим синхронизации: только новые посты / история + новые / только публикация.

> **Демо-канал:** введите `@demo-kanal` — загрузится набор тестовых постов для знакомства с платформой.

## 4. Базовый рабочий сценарий

1. **Создайте пост** — нажмите кнопку «+» в ленте или в sidebar.  
   Откроется редактор (Composer).
2. **Напишите текст** и при необходимости добавьте медиафайлы или прикрепите контент из другого поста.
3. **Попросите AI** помочь улучшить текст — нажмите иконку AI в редакторе.
4. **Запланируйте** или **опубликуйте** пост.
5. Просматривайте **статистику** в разделе Аналитика после публикации.
