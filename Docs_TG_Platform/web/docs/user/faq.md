# FAQ — Частые вопросы

## Общие

**Нужен ли бэкенд для работы платформы?**  
В текущей версии платформа работает полностью на фронтенде с имитацией API (MSW). Данные хранятся в памяти браузера. При перезагрузке сессия сбрасывается. Полноценный бэкенд запланирован.

**Можно ли использовать без Telegram-аккаунта?**  
Да. Большинство функций (черновики, чаты, заметки, AI) работают без подключения Telegram. Для публикации и синхронизации истории Telegram необходим.

**Данные сохраняются между сессиями?**  
Пока нет — это ограничение демо-версии. В продакшн-версии данные будут сохраняться на сервере.

## Посты и лента

**Как изменить порядок черновиков?**  
Перетащите карточку черновика в нужное место — порядок сохранится.

**Как запланировать пост?**  
Через контекстное меню карточки → «Запланировать», либо из рабочей области поста.

**Можно ли восстановить удалённый пост?**  
Нет, удаление постов необратимо. Будьте внимательны.

---

## AI и чаты

**Почему AI не отвечает?**  
Проверьте настройки AI-моделей — должна быть задана хотя бы одна активная LLM-модель с корректным API-ключом.

**Как AI учитывает контекст поста?**  
Когда вы ведёте диалог из рабочей области поста, AI автоматически получает в контекст: профиль канала, рубрику поста и заметки с флагом AI.

**Что такое Multi-response?**  
Режим, при котором AI генерирует несколько вариантов ответа одновременно от разных моделей. Включается в Настройках → AI.

## Заметки

**В чём разница между глобальной заметкой и заметкой к посту?**  
Глобальные заметки — общие записи без привязки к посту (идеи, шаблоны). Заметки к посту — записи в контексте конкретного поста и участвуют в AI-контексте чатов к нему.

**Что даёт флаг AI на заметке?**  
AI-заметки включаются в контекст запросов к AI при работе над постом. Удобно для хранения источников, фактов, требований.

## Настройки

**Где взять API ID и API Hash для Telegram?**  
На [my.telegram.org](https://my.telegram.org) → «API development tools». Необходимо войти через SMS-код.

**Как удалить аккаунт?**  
В текущей версии удаление аккаунта недоступно через интерфейс. Обратитесь к администратору.
