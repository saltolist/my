# Заметки

Заметки — свободное пространство автора: черновики, идеи, база знаний. Платформа не диктует сценарий использования.

> **UI reference:** [pages.md](../ux/pages.md) (§5 `note`, §7 `notes`, §4 `post` режим `notes`) · [wireframes](../ux/wireframes/04-note.md).

---

## Два уровня

| Уровень | Где создаются / видны | Привязка |
|---|---|---|
| **Глобальные** | каталог `notes`, сайдбар «Заметки», страница `/note/global/{id}/` | канал |
| **Локальные** | режим `notes` в пространстве поста, каталог `notes` (scope `Локальные` / `Все`), сайдбар | конкретный пост |

Локальные заметки в общем каталоге и в сайдбаре — **штатное поведение** reference client, не «упрощение».

---

## Каталог `notes`

**Шапка:** `Заметки`; поиск; dropdown scope **`Все` / `Глобальные` / `Локальные`** (стартовый — `Все`); `← Назад` → `home`.

**FilterToolbar:** фильтр **`Все` / `В контексте ИИ` / `Не в контексте`** + **`+ Новая заметка`** (в scope `Все` и `Глобальные`).

Карточки: превью, переключатель **`● ИИ` / `○ ИИ`**, удаление.

---

## Страница заметки

Режимы `view` / `edit`. Хлебные крошки:

- глобальная: `Заметки / Название`;
- локальная: `Лента / Название поста / Название` (средняя часть — ссылка на пост).

Меню `•••`: **`Учитывать в ИИ` / `Не учитывать в ИИ`**, удаление.

Редактор: заголовок (textarea), тело с embed-чипами, блок «Вложения», drag-and-drop файлов и перестановка embed-чипов в `edit`.

**Отдельной кнопки «Обсудить с ИИ» на странице заметки нет** — диалог с ИИ идёт через глобальный или локальный чат.

---

## AI-флаг («учитывать в ИИ»)

Есть **и у глобальных, и у локальных** заметок (карточка в каталоге, карточка в режиме `notes` поста, меню `•••` на странице).

| Флаг | Глобальный ИИ | Локальный ИИ поста |
|---|---|---|
| глобальная заметка **включена** | да | да (наследуется) |
| глобальная заметка **выключена** | нет | нет |
| локальная заметка **включена** | — | да (свой пост) |
| локальная заметка **выключена** | — | нет |

Фильтр каталога «В контексте ИИ» / «Не в контексте» опирается на тот же флаг.

---

## Режим `notes` в пространстве поста

Сетка локальных заметок (3 в ряд на desktop), поиск, **`+`** на карточке создаёт заметку, **`● ИИ` / `○ ИИ`** и **`🗑`** на карточке.

---

## Примеры использования (не ограничения)

- **Локальные** — черновик абзаца, идея под конкретный пост, альтернативный текст.
- **Глобальные с AI-флагом** — факты и исследования для контекста канала.
- **Глобальные без флага** — личные записи, ссылки «на потом», без участия в контексте ИИ.
