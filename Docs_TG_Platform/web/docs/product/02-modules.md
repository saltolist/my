# Модули платформы (web-клиент)

Документ фиксирует модули reference web-клиента ([`web-legacy`](../../../web-legacy/)). Детали экранов — [pages.md](../ux/pages.md), компоненты — [components/](../ux/components/).

**UX = legacy 1:1.** Product-доки объясняют модель и контекст; если текст расходится с `pages.md` — правим документацию, а не считаем это «scope gap».

## Структура навигации

```
TG Platform
├── [✦ лого] / [Глобальный чат]  ← home (+); сворачивание — отдельная кнопка rail
├── Аналитика канала
├── Лента                        ← все посты: опубликованные / отложенные / черновики
│   └── Пространство поста       ← локальные чаты + заметки + комментарии поста
│       ├── режим chat           ← открытый локальный чат
│       ├── режим chats          ← список локальных чатов (поиск)
│       ├── режим notes          ← сетка локальных заметок (поиск)
│       ├── режим comments       ← комментарии (только опубликованные; поиск)
│       └── Страница заметки     ← локальная заметка
├── Заметки ▾                    ← каталог + недавние в сайдбаре
│   └── Страница заметки         ← глобальная или локальная
├── Чаты ▾                       ← каталог + недавние в сайдбаре
└── Профиль канала
    ├── Настройки                ← стартовая вкладка
    ├── Канал                    ← база знаний (core, voice, rules, rubrics)
    └── Аналитика платформы
```

---

## Глобальный чат

- `home` — новый чат;
- `gchat` — сохранённый чат.

**Реализовано:**

- composer с LLM, Web Search, мультиответом;
- attach: `Прикрепить файл`, `Медиа из прикреплённых постов`;
- **пост через `@` mention**, не через attach-меню;
- удаление из меню `•••`.

---

## Лента

Секции: `Опубликованные` (группы по дням) → `Отложенные` → `Черновики`.

- composer внизу, `+` → file picker;
- ширина: Компьютер / Планшет / Телефон;
- карточка: статус SVG, метрики, `PostCommentsRow` отдельной строкой;
- drag-handle 6-точечный у черновиков;
- рубрика на карточке не показывается.

---

## Пространство поста

Режимы: `chat`, `chats`, `notes`, `comments`.

- `+ Новая заметка` / `+ Новый чат` — в FilterToolbar;
- `← Назад` — browser history;
- меню: Новый чат, Новая заметка + publish/schedule/reschedule/delete;
- chat: PostCardToolbar (copy/edit), composer без «Прикрепить пост»;
- notes/chats: фильтр AI-контекста в FilterToolbar.

---

## Чаты

Dropdown scope в шапке. `Новый чат` на вкладках Все/Глобальные. Локальный чат → клик по карточке (без `→ пост`).

---

## Заметки

Dropdown scope. FilterToolbar с AI-фильтром. Toggle «В контексте» / «Не в контексте».

---

## Аналитика канала

5 периодов, trend chart, metric bars, reactions, heatmap, top posts — см. [pages.md](../ux/pages.md#8-analytics).

## Профиль

Стартовая вкладка `Настройки`. Канал — core / voice / rules / rubrics. Настройки — тема, 5 групп моделей + Telegram + омнибот. Аналитика платформы — models chart, usage, activity.

Контент полей — [05-channel-profile.md](./05-channel-profile.md); UI — [profile-settings.md](../ux/components/widgets/profile-settings.md).
