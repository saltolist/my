Каждый ADR описывает контекст, рассмотренные варианты и обоснование выбора.

## Формат ADR

```markdown
# ADR-NNN: Название

## Статус
Принято / Отклонено / Устарело

## Контекст
Почему возникла необходимость принять решение.

## Рассмотренные варианты
- Вариант A
- Вариант B (выбранный)

## Решение
Что именно было выбрано и как реализовано.

## Последствия
Что изменилось, на что влияет это решение.
```
